# 模型性能评估
## 1. TM score
TM-score（Template Modeling Score）： 模板建模分数
蛋白质结构*拓扑折叠*的相似度指标， 衡量“预测结构”与“天然/模板结构”的整体折叠匹配度
范围 **0~1**，越接近1越相似
$$ TM-score = \max \frac{1}{L} \sum_{i=1}^{L_{ali}} \frac{1}{1+(d_i/d_0)^2} $$
- **$L$**：目标蛋白（或结构域）全长 
- **$L_{\text{ali}}$**：对齐残基数
- **$d_i$**：第$i$对对齐Cα原子的空间距离（Å）
- **$d_0$**：长度相关归一化因子（消除蛋白长度偏差）
	- $d0​(L)=1.24\sqrt[3]​{(L−15)}−1.8​$

### 特点
1. 重全局、轻局部：小距离误差权重远大于大误差。 
2. 长度无关：通过$d_0$归一化，不同长度蛋白/结构域可直接比较。 
3. 无硬阈值：不设距离截断（如GDT的1/2/4/8 Å），利用所有对齐残基
### 分数阈值 
- ≥0.8：完全一致折叠（高精度模型）
- 0.5~0.8：相同折叠家族（正确拓扑，结构域边界可靠） 
- 0.3~0.5：部分相似（局部折叠对，全局拓扑不同） 
- <0.3：随机无关（无结构同源性） 
## 2. t检验
*作用*： 比较两个模型的性能是否有显著区别
*要求*： 样本相同
### 计算
设一共 n 个样本（比如 n 个蛋白结构域）：
- 模型 A 分数：x1​,x2​,…,xn​
- 模型 B 分数：y1​,y2​,…,yn​
1. **差值**：： $d_i​=x_i​−y_i​(i=1,2,…,n)$
2. 计算差值的均值$\overline{d}=\frac{1}{n}\Sigma_{n=1}^nd_i$
3. 标准差$S_d = \sqrt{\frac{\Sigma_{i=1}^n(d_i-\overline{d})^2}{n-1}}$
4. 标准误:$SE=\frac{S_d}{\sqrt{n}}$
5. t值计算:t=\frac{overline{d}{SE}
## 3. $MAE_n$
平均绝对误差
$$MAE_n = \frac{\Sigma_{i, j}|d_{ij}^{pred}-d_{ij}^{true}|}{\Sigma_{i, j}{d_{ij}}^{true}}$$
## 4. RMSD
均方根误差
$$RMSD = \sqrt{\frac{1}{N^2}\Sigma_{i,j}(d_{ij}^P-d_{ij}^t)^2}$$
