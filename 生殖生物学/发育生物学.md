# 杂
* 前口动物，后口动物
	* 前口动物： 胚孔发育成口
	* 后口动物： 胚孔发育成肛门
* 生殖质： 特定的细胞质成分， 位于卵或胚胎特定部分， 决定原始生殖细胞的分化
* 卵裂没有细胞生长的间期
* 核型： 生物体细胞中所有染色体的大小， 形状， 数量特征的总和
* 性腺
	* 定义： 人体中兼具生殖和内分泌功能的腺体
	* 类型
		* 睾丸
		* 卵巢
* 分泌方式
	* 旁分泌： 邻近细胞， 不进入血液
		* 近分泌： 特殊的旁分泌， 邻近细胞
	* 自分泌： 作用于自身
	* 外分泌： 管道， 体表或管腔
	* 内分泌： 无管道
* 肾上腺和性腺都起源于生殖脊

# 生殖细胞发生
* 生殖细胞发生过程
	* 原始生殖细胞——生殖母细胞——精原干细胞——精原细胞——精细胞
* 原始生殖细胞: Primordial Germ Cels, PGCs
	* 性别分化以前的生殖细胞， 具有双潜性（能分化成精子或卵子）
	* 迁移到生殖嵴（生殖腺原基）分化成生殖细胞
	* 来源及迁移
		* 生殖质： 两栖类动物卵子中的特殊细胞质， 蛋白质和RNA组成。分裂后的子代细胞中含有这种细胞质的细胞将发育成原始生殖细胞
			* 证明实验： 果蝇卵的尾部细胞质的移植实验
		* 线虫
			* P细胞：含有P颗粒的细胞 
				* 线虫的生殖质称为P颗粒
		* 果蝇
			* 极细胞： 含有极质的细胞
				* 极质： 果蝇的生殖质
				* 极粒： 极质中的致密小体， 核糖核蛋白构成（mRNA+蛋白质）
			* 胚胎后端的极细胞——通过原肠作用到达中肠后部——穿过肠壁和中胚层——生殖腺
		* 斑马鱼
			* 生殖质存在受精卵多个位置， 随卵裂像分裂沟聚集， 定位在分裂沟的远端（未来生殖嵴原基的位置）
			* 受精卵中生殖质位于四个位置——体节胚胎时期原始生殖细胞在躯干两边形成细胞系——趋化因子SDF1a的趋化作用下到达相应位置——与周围体细胞形成性腺
		* 两栖类
			* 爪蟾
				* 生殖质呈小岛形式， 靠近植物极
				* 囊胚期原始生殖细胞位于囊胚腔底部
			* 有些两栖类没有明确生殖质
			* 迁移路线
				* 植物极——囊胚腔底部——内胚层表面——肠后部聚集——中肠上部生殖嵴
		* 鸟类
			* PGCs起源于胚盘上胚层明区中央——原肠作用——原条期明区前部边缘形成生殖新月区——新月区形成血管——PGCs进入血管——血液循环——生殖嵴
		* 哺乳动物
			* 小鼠
				* 碱性磷酸酶阳性鉴别PGCs
			* 胚胎后端——后肠内卷下进入后肠内胚层——主动到达生殖脊， 途中进行细胞增殖
	* PGC迁移机制及影响因素
		* 机制
			* PGC的运动能力是其固有属性（伪足等）， 但具有时效性
			* 生殖脊释放趋化因子引导生殖脊
			* 形成伪足相互接触
			* 迁移过程中形成纤连蛋白受体附着体细胞
				* PGC迁移途中SCF（干细胞生长因子）表达量最高
				* PGC表达SCF受体C-kit
			* PGC细胞间形成网络连接（伪足）， 引导后出发的细胞移动
* 精卵子发生选择 ^5a7db6
	* PGCs进入生殖嵴先进行有丝分裂， 随后根据性别不同进入不同发育阶段
		* 雌性： 出生前进入双线期， 停滞——性成熟时继续减数分裂
		* 雄性： 出生前停滞G1期， 性成熟后某些有丝分裂， 某些减数分裂
- 精子发生 ^8a3aae
	- 定义： 原始生殖细胞形成精原细胞到精子并排出体外的过程
	- 过程
		- 精原细胞增殖
		- 精母细胞减数分裂
		- 精子变形
	- 精原细胞的类型
		- Ap型：浅色型
			- 分化成B型
			- 核卵圆状或椭球盘型
			- PAS阴性
		- Ad型：黑色型
			- 精原干细胞
			- 分化成一个Ad型一个Ap型
			- 染色深
			- PAS阳性
		- B型
			- 分化成精母细胞
	- PGCs进入生殖腺原基（发育成睾丸）， 进入性索停留。性索发育成生精小管（曲细精管）。管上皮细胞分化成支持细胞
	- 细胞质桥
		- Ad型精原细胞分裂开始，为细胞不完全分裂。
		- 细胞间通过细胞质桥连接， 形成合胞体， 进行物质和信息交流
		- 实现每群细胞同步成熟
	- 过程
		- 增殖
		- 减数分裂
			- 细线期， 偶线期， 粗线期， 双线期， 终变期
		- 塑形
			- 生精小管中的变化
				- 高尔基体形成顶体泡——顶体
					- 高尔基体形成顶体泡，顶体泡融合形成顶体囊
					- 顶体囊内含顶体颗粒。
					- 顶体囊将水分排出， 形成双层膜，内含水解酶
					- 顶体面向管壁
					- 大型哺乳动物卵圆形， 小鼠大鼠镰刀形
				- 鞭毛
					- 中心粒微管延长——轴丝
						- 外周有纤维和纤维鞘
					- 线粒体整合入鞭毛——线粒体鞘
						- 线粒体伸长， 迁移到中段， 围绕中央轴丝螺旋排列
				- 体形变小
					- 多余胞质丢弃——残体
					- 染色体凝缩， 精核重构（鱼精蛋白代替核小体与DNA结合， 大量， 非全部）——精子头
						- 鱼精蛋白是一种核精蛋白
					- 精膜上卵子结合蛋白合成
			- 精子离开睾丸生精小管后形态结构成熟， 在附睾尾具有受精和定向运动能力
	- 调控机制
		- 激素调控
			- 下丘脑——促性腺激素释放激素（GnRH）
			- 脑下垂体——性腺激素（LH, FSH）
			- 睾丸
				- LS受体——生精小管间质细胞
					- 胆固醇转变为睾酮
						- 参与增殖， 减数分裂等过程
						- 没有睾酮无法完成精子发生
						- 肾上腺也能产生睾酮
				- FSH受体——生精小管支持细胞
					- 合成分泌蛋白为生殖细胞输送营养
					- 合成抑制素， 与睾酮进行负反馈调节
		- 旁分泌调节
			- 睾酮， 生长因子等
		- 基因调控

# 受精
* 定义： 两性生殖细胞融合， 新个体
* 类型：自受精， 异受精
* 意义
	* 性状稳定
	* 变异
	* 发育开端
* 方式
	* 体外受精： 无脊椎， 低等脊椎
		* 海胆——棘皮动物， 模式生物
			* 精卵识别
				* 卵母细胞分泌趋化因子， 形成微环境， 吸引精子
				* 海胆
					* 趋化因子： 呼吸活化肽
			* 卵——见pdf图片
				* 有两种膜
					* 卵黄膜： 与卵膜相连， 纤维物质， 可与海胆精子顶体融合
					* 卵胶膜： 透明， 主要成分为糖蛋白(呼吸活化肽)
						* 卵子排出体外后， 胶膜缓慢溶于水， 引起精子趋化， 诱发顶体反应

- Volcher Coiter： 卵子从卵巢形成
- William Harvey：变态在昆虫中普遍存在
- 冯贝尔：冯贝尔法则
	- 总结：  不同纲脊椎动物的早期胚胎是极其相似的，在胚胎发育过程中，门的性状最先出现，随后才依次出现目、科、属、种的性状。这种规律被称为贝尔法则
	- （1）每一较大动物类群的共同性状比专门性状形成得较早；
	- （2）先出现最共同的性状，然后出现较专门的性状，最后才出现最专门的性状；
	- （3）不同纲动物的胚胎最初是相似的，在以后的发育中才逐步分歧。
