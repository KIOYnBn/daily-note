##### add_subgridspec
```python
{
fig = plt.figure(layout="constrained")
gs0 = fig.add_gridspec(1, 2)

gs00 = gs0[0].subgridspec(2, 2)
gs01 = gs0[1].subgridspec(3, 1)

for a in range(2):
    for b in range(2):
        ax = fig.add_subplot(gs00[a, b])
        annotate_axes(ax, f'axLeft[{a}, {b}]', fontsize=10)
        if a == 1 and b == 1:
            ax.set_xlabel('xlabel')
for a in range(3):
    ax = fig.add_subplot(gs01[a])
    annotate_axes(ax, f'axRight[{a}, {b}]')
    if a == 2:
        ax.set_ylabel('ylabel')

fig.suptitle('nested gridspecs')
}
```

## 图形参数
* figsize(宽度， 高度）： 设置图形大小， 单位为英寸
* dpi： 每英寸渲染图形的像素数， 通常设置为与图形系统相同
* facecolor： 填充色
* edgecolor： 边框色
* linestyle： 设置线条样式
	* -： 实现
	* --： 虚线
	* -.：点划线
	* 、：： 点线
* linewidth： 线宽（数值）
* frameon： 边框显示， 布尔值
* gridspec_kw： 设置宽高比
* layout： 布局引擎。调整内部Axes放置刻度标签重叠， 对齐Axes
	* constrained： 约束布局， 自动调整子图间距和坐标轴位置， 防止重叠， 适用于多子图复杂布局
	* compressed：压缩布局， 比tight更激进的压缩间距， 通常不使用
	* tight：紧凑布局： 手动调整子图与画布边缘间距， 压缩子图间空白
		* pad： 边缘留白比例
		* h_pad/w_pad: 垂直/水平间距
---

```python
{
gs_kw = dict(width_ratios=[1.4, 1], height_ratios=[1, 2])
fig, axd = plt.subplot_mosaic([['upper left', 'right'],
                               ['lower left', 'right']],
                              gridspec_kw=gs_kw, figsize=(5.5, 3.5),
                              layout="constrained")
for k, ax in axd.items():
    annotate_axes(ax, f'axd[{k!r}]', fontsize=14)
fig.suptitle('plt.subplot_mosaic()')
}
```

### 参数
* loc： 设置关键词设置图例位置
* bbox_to_anchor: 与loc结合定位于图例的框
	* 四元组浮点数时： (x,y, width, height)
	* 二元素浮点数时： (x, y)
* bbox_transform: 边界框的变换， 使用axes的transAxes变换
	* 示例
		* bbox_transform=fig.transFigure
* ncols: int， 图例列数
* prop： 图例字体
* fontsize： 字体
* labelcolor： str/list
* numpoints： int， 标记点数量
* facecolor： 图例背景颜色
* edgecolor：
* mode： {"expand", None}
	* 若为expand， 图例水平拓展填充axes区域
* borderpad： float， 空白区域比例
