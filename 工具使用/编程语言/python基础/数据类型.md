# 容器类型
1. list
	* 添加
		* 末尾添加： append
		* 指定位置添加: insert
		* 批量添加： extend
	* 删除： 
		* 按索引： del
		* 按值： remove
		* 删除并返回： pop
	* 长度： len
	* 排序： sort
	* 反转： reverse
	* 统计次数： count
2. tuple
	* 长度： len
	* 统计： count
	* 索引： index
3. dict
	* 取值： get
	* 删除： del
	* 删除并范围： pop
	* 删除最后的键值对： popitem
4. set
	* 添加： add
	* 批量添加： update
	* 删除： remove/discard
	* 随机删除： pop
	* 运算： 
		* &： 交集
		* |: 并集
		* -： 差集
		* ^: 对称差集
