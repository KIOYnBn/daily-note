# 文件打开与关闭
* 文件打开： open
	* `file_object = open(file, mode='r', encoding=None)`
	* 参数： 
		* file_path: str
		* mode: str
		* encoding: utf-8/gbk
			* 中文文件必须要utf-8
* 文件关闭： close
	* `file_object.close`

# 读取
* read(size:int): size为字符串数或字节数.默认为读取全部
	* `content = file_object.read(size)
* readline: 逐行读取， 包括换行符。文件末尾为空字符串
* readlines： 读取所有行， 返回列表
* for循环： `for index, line in enumerate(file_object):`
# 写入
* write(str)
* writelines(iterable)
---
# 文件指针
* tell(): 返回当前文件指针位置（字节）
* seek（offset, whence)： 移动指针
	* offset:int， 正数为向后移动， 负数为向前移动
	* whence： 
		* 0： 文件开头
		* 1： 当前位置
		* 2： 文件末尾

# os库
* os.mkdir(path): 创建一级目录
* os.mkdirs(path): 创建多级目录
* os.listdir(path)： 返回所有文件和子目录
* os.walk: 返回目录下所有目录与子目录， 返回三元组(当前路径， 子目录路径， 文件列表)
* os.rmdir(path)： 删除单一级目录
* os.rmdirs(path): 删除多级目录
* os.rename(old_name, new_name)： 重命令
* os.remove(path)： 删除文件， 不删除目录
* os.path.exists(path)： 判断文件是否存在
* os.path.isfile(path)
* os.path.isdir(path)
* os.path.join(path1, path2)
