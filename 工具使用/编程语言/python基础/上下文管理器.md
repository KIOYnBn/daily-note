# 基础知识
* 上下文管理器需要包含enter和exit两个基本函数
* with进入时， 调用enter；结束时， 调用exit
* exit参数解析
	* exc_type: 异常类型
	* exc: 异常对象（信息）
	* tb： traceback调用栈欣喜
	* 返回True， 则会抑制异常；False， 异常继续抛出

# 自定义with上下文管理器
```python
import time
class Timer:
	def __enter__(self):
		self.start = time.perf_counter()
		return self
	def __exit__(self, exc_type, exc, tb):
		self.end = time.perf_counter()
		self.elapsed = self.end-self.start
		print(f"耗时: {self.elapsed:.4f}s")
		return False
def slow_function():
	time.sleep(1)
		
with Timer() as t:
	slow_function()
```

# contextmanager便携用法
* 示例
```python
from contextlib import contextmanager
import time

@contextmanager
def timer()
	start_time = time.perf_counter()
	try:
		# 相当于enter
		yield 
	finally:
		# 相当于exit
		end_time = time.perf_count()
		print(f'Time:{end_time-start_time:.4f}')

with timer():
	text:str = ''
	for i in range(100):
		text += str(i)
		
```
