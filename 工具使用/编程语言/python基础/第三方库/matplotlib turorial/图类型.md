# 图类型总结

| 代码         | 图类型       | 关键参数                                 |
| ---------- | --------- | ------------------------------------ |
| plot（）     | 2d关联分析    | x，y： 数据数组；linestyle： 线型；marker： 标记类型 |
| scatter（）  | 离散数据      | x，y： 数据坐标；c： 颜色映射字段；s：点的大小           |
| bar（）      | 垂直/水平柱状图  | x： 柱体位置， height： 柱高；width： 柱宽        |
| step       | 阶梯图       | where： 阶梯方向-pre/post/mid             |
| pcolormesh | 彩色网格图     | shading： 着色方式                        |
| contour    | 等高线图      | levels： 等高线层级；colors： 等高线颜色          |
| quiver     | 矢量场箭头图    | u，v： 矢量分量；scale： 缩放因子；               |
| streamplot | 流线图       | density： 流线密度；linewidth：流线宽度         |
| imshow     | 图像或矩阵数据   | origin： 坐标原点；interpolation： 插值方法     |
| hist       | 一维直方图     | bins： 边界；density： 归一化                |
| errorbar   | 带误差线的数据点  | yerr： y轴误差；fmt： 数据点样式                |
| hist2d     | 二维直方图     | bins： xy轴分箱数， cmin： 最小计数阈值           |
| boxplot    | 箱线图       | positions：箱体位置；showfliers： 显示离群值     |
| violinplot | 小提琴图      | scale： 缩放                            |
| tricontour | 三角剖分的等高线图 | tri： 三角剖分对象                          |
| tripcolor  | 三角剖分的彩色图  | shading： 着色方式                        |
 ## 绘制时间图

```python
{
from matplotlib.dates import ConciseDateFormatter

fig, ax = plt.subplots(figsize=(5, 2.7), layout='constrained')
dates = np.arange(np.datetime64('2021-11-15'), np.datetime64('2021-12-25'),
                  np.timedelta64(1, 'h'))
data = np.cumsum(np.random.randn(len(dates)))
ax.plot(dates, data)
ax.xaxis.set_major_formatter(ConciseDateFormatter(ax.xaxis.get_major_locator()))
}
```

# 色图：pcolormesh
* 颜色条： colorbar
	* 参数
		* pcm
		* ax
		* shrink： 缩放
		* location: bottom, right, left
		* pad
		* label

```python
{
from matplotlib.colors import LogNorm

X, Y = np.meshgrid(np.linspace(-3, 3, 128), np.linspace(-3, 3, 128))
Z = (1 - X/2 + X**5 + Y**3) * np.exp(-X**2 - Y**2)

fig, axs = plt.subplots(2, 2, layout='constrained')
pc = axs[0, 0].pcolormesh(X, Y, Z, vmin=-1, vmax=1, cmap='RdBu_r')
fig.colorbar(pc, ax=axs[0, 0])
axs[0, 0].set_title('pcolormesh()')

co = axs[0, 1].contourf(X, Y, Z, levels=np.linspace(-1.25, 1.25, 11))
fig.colorbar(co, ax=axs[0, 1])
axs[0, 1].set_title('contourf()')

pc = axs[1, 0].imshow(Z**2 * 100, cmap='plasma', norm=LogNorm(vmin=0.01, vmax=100))
fig.colorbar(pc, ax=axs[1, 0], extend='both')
axs[1, 0].set_title('imshow() with LogNorm()')
```
