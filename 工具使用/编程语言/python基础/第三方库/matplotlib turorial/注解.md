# 参数

* text: str
* xy: 数据位置
* xytext： 文本位置
* arrowprops： 箭头属性设置
	* 参数
		* width: 箭头宽度
		* frac： 头部所占百分比
		* headwidth：头部的尾部宽度
		* shrink： 箭头尾部和头部相对目标点偏移量
		* kwargs：pollygon参数
	* 自定义箭头： dict（[[#arrowstyle|arrowstyle]]， [[#connectionstle]]）
* va： 垂直方向位置
* ha： 水平方向位置
* [[#coords|textcoords]]： 
	* 文本的坐标系
	* 相对文本的偏移量
* [[#coords|xycoords]]： 
	* 数据点的坐标系
	* 相对数据点偏移量
* [[#bbox|bbox]]： 画框
	* 访问： get_bbox_patch
	* 属性设置： set_boxstyle
