# 条件控制语句
* if语句
```
if 条件表达式:
    满足条件时执行的代码块（注意缩进）
    

if 条件表达式:
    满足条件时执行的代码块
else:
    不满足条件时执行的代码块
    

if 条件1:
    满足条件1执行的代码块
elif 条件2:
    不满足条件1但满足条件2执行的代码块
...（可多个elif）
else:
    所有条件都不满足时执行的代码块

结果1 if 条件 else 结果2
```

# 循环语句
* for语句
```

for 变量 in 可迭代对象（列表/字符串/range等）:
    每次循环执行的代码块
```

* while循环
```

while 条件表达式:
    条件满足时执行的代码块
    （需设置终止条件，避免死循环）
```
