| 方法             | 作用                     | 适用场景           |
| -------------- | ---------------------- | -------------- |
| `json.dumps()` | 将Python对象转为JSON字符串     | 内存中处理JSON数据    |
| `json.dump()`  | 将Python对象写入文件（JSON格式）  | 保存数据到JSON文件    |
| `json.loads()` | 将JSON字符串转为Python对象     | 解析接口返回的JSON字符串 |
| `json.load()`  | 从文件读取JSON数据并转为Python对象 | 读取JSON文件       |
