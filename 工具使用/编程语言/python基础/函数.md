* 函数定义与调用
```

def 函数名(参数列表):
    """文档字符串（可选，说明函数功能）"""
    函数体（实现功能的代码）
    [return 返回值]  # 可选，无return则默认返回None
    
函数名(参数值列表)
```

* 参数类型
	* 位置参数： 
	* 关键词参数： “参数名=值”传值
	* 默认参数： 定义函数时给参数指定默认值
		* 默认参数必须放在**位置参数之后**（否则语法报错）。
	* 可变位置参数： `*args`
	* 可变关键词参数: `**kwargs`
	* 参数解包：
		* 列表解包： *
			* `nums = [1, 2]; result1 = add(*nums)  # 等价于add(1, 2)`
		* 字典解包： **
			* `num_dict = {"a": 3, "b": 4};result2 = add(**num_dict)  # 等价于add(a=3, b=4)`
* 匿名函数: lambda
	* `lambda 参数列表: 表达式`（表达式结果就是返回值，无需写return）。
# 闭包
* 定义： 
	* “嵌套函数中，内部函数引用了外部函数的变量，且外部函数返回了内部函数，此时内部函数及其引用的外部变量共同构成一个「闭包」”。
* 作用： 保存外部变量状态
* 示例
```python

# 外部函数：接收一个初始值 base
def outer_func(base):
    # 定义内部函数（嵌套函数）
    def inner_func(num):
        # 内部函数引用了外部函数的变量 base（核心条件2）
        return base + num
    
    # 外部函数返回内部函数对象（核心条件3，不执行内部函数）
    return inner_func

# 1. 调用外部函数，返回内部函数对象，赋值给 add5
add5 = outer_func(5)
# 2. 调用内部函数，此时 outer_func 已执行完毕，但 inner_func 仍能访问 base=5
print("5 + 10 =", add5(10))  # 15
print("5 + 20 =", add5(20))  # 25

# 3. 重新调用外部函数，创建新的闭包（base=10）
add10 = outer_func(10)
print("10 + 30 =", add10(30))  # 40
```
## 装饰器
### 函数装饰器
* 作用： 
	1. 在不改变原函数的前提下， 添加功能
	2. 功能复用
* 示例
```python
def outer(func):
	def x(*args, **kwargs):
		print("start send")
		value = func(*args, **kwargs)
		print("end send")
		return value
	return x
	
@outer
def send_wechat(body):
	print("微信", body)
	return 100
	
res = send_wechat("你好")
print(res)
```

### 类装饰器
* 原理： 类加入可调用函数__call__后， 可以像函数一样调用
* 示例
```python
class Deco:
	def __init__(self, char, num):
		self.char = char
		self.num = num
		
	def __call__(self, f):
		def newFunc(*args, **kwargs):
			print(self.char*self.num)
			res = f()
			print(self.char*self.num)
			return res
		return newFunc
		
@Deco('=', 20)
def func():
	print('hellowrld')
func()
```
