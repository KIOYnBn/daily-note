# 全局变量
* 定义在函数外或define的变量
	* 全局变量不初始化时， 会默认给值
		* int默认为0， 指针默认为None
```c
#include <stdio.h>
#define WEATHER 33

int global_var = 222;

int main(void) {

	printf("%d, %d", WEATHER, global_var);
	return 0;
	
}
```

# 静态变量
* 用关键词static修饰的变量
* 特点： 只初始化一次
* 用途： 函数内局部变量存储
* 类型： 
	* 局部静态变量：
	* 全局静态变量： 
		* 正常全局变量： 可以全文件访问
		* 全局静态变量： 只能该文件访问
```c
#include <stdio.h>

static int num1 = 0;

void print_static_var(void);

int main(void) {

	int circulation_num = 3;
	for (int index=0; index<circulation_num; index++) {
		print_static_var();
	}
}

void print_static_var(void) {
	static int num2 = 0;
	printf("the num2 is %d\n", num2);
	num2++;
}
```

# 外部变量
* 用关键词extern修饰的变量
* note： static不能用extern修饰
* 示例
	- 文件run_main.c
```c
#include<stdio.h>

int global_var = 1;

int main(void) {
	return 0;
}
```
* 文件helper.c
```c
#include<stdio.h>

extern int global_var;

void print_num(void) {
	printf("%d", global_var);
}
```
