* 注意事项
	* 数组初始化的长度不能用变量表示， const修饰的也不行， 只能通过define
	* 定义数组是必须初始化, 不能`uint32_t arr[10];`, 可以`uint32_t arr[10] = {}`或者`uint32_t arr[10] = {0}`
	* c语言的数组索引与python不同， 不可以用负数
```
#include <stdio.h>
#include <stdint.h>
int main(void) {
	uint32_t arr[10] = {};
	uint32_t arr1[10] = {0};
	uint32_t two_dimension_arr[2][3] = {
		{1, 2, 3},
		{4, 5, 6},
	};
	// 隐式确定数组大小
	uint32_t two_dimension_arr1[][3] = {
		{1, 3, 4},
	};
	
}
```
