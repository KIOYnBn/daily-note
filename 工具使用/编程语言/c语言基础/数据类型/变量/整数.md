* 当前工程中， 通常使用stdint.h中的int8_t, int16_t, int32_t, int64_t。而不用short, int, long
	* int32后， uint32需要加U, long类型加L
* 当前工程的格式化输出int， 通常使用inttypes.h的`PRId8`, `PRId16`等， 而不是%d等
* 整数类型： 
	* 十进制： 数字0~9开头
	* 八进制： 数字0开头
	* 十六进制： 数字0x开头， 不分区字母大小写
* 后缀： 
	* u： unsigned
	* l: long: 
	* ll: long long

|平台/模型|`int` 大小|`long` 大小|`long long` 大小|
|---|---|---|---|
|Windows (LLP64)|4 字节（32位）|4 字节（32位）|8 字节（64位）|
|Linux/macOS (LP64)|4 字节（32位）|8 字节（64位）|8 字节（64位）|
|16位系统（罕见）|2 字节（16位）|4 字节（32位）|8 字节（64位）|


```c
#include <stdio.h>

int main() {
	int dec_int = 28;
	short s_int = 28;
	int oct_int = 027;
	int x_int = 0x2A;
	unsigned dec_uint = 28u;
	long dec_long = 28l;
	long long dec_llong = 28ll;
	unsigned long long dec_ullong = 28ull;
	printf("i: %d, h:%hd, o: %#o, x: %#x, u: %u, l: %ld, ll: %lld, ull:%llu", dec_int, s_int, oct_int, x_int, dec_uint, dec_long, dec_llong, dec_ullong);
	return 0;
}
```
