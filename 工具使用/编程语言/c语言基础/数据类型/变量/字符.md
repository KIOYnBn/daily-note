# 基本概念
* 字符： 由`''`包裹
* 字符串： 由`""`包裹， 表示一个字符序列
	* 多行字符串的末尾用`\`连接
	* 最长为509/2048个字符
* 多字节： 由一个或多个字节的序列构成的字符
	* 如`"hello"`占6字节（5个字母+\0）
	* 用于多语言： 如`"中"`在内存里可能2/3字节
* 宽字符： 宽度始终为16位（2字节）的字符代码
	* 占用空间大， 将文本变成数组
	* 适用于国际字符集unicode
	* 类型为： wchar_t
	* `L"hello"` -> `{L"h", L"e", L"l", L"l", L"o", 0}`
* 总结
```
`'a'` 是一个普通的 `char` 类型字符常量。
    
`L'a'` 是一个 `wchar_t` 类型的宽字符常量。
    
`"hello"` 是一个 `char` 数组（多字节字符串）。
    
`L"hello"` 是一个 `wchar_t` 数组（宽字符串）。
```

# 特殊字符
## 空白字符
* 包括`空格， 制表符， 换行符， 回车符， 换页符， 垂直制表符， 注释`
* C编译器会忽略空白字符
---
## 转义序列
* 某些字符常量需要通过转义序列表示——前面加一个`\`。
	* 单引号： `'`
	* 反斜杠： `\`
	* 换行符
* 八进制转移序列： `\101`
* 十六进制转义序列： `\x410`

# 字符
```c
#include <stdio.h>
int main(void) {

	// ---------- 字符常量 ----------
    char c1 = 'A';             // 普通字符常量，类型 char，ASCII 值为 65
    char c2 = '\n';            // 转义字符常量，换行符
    wchar_t wc = L'中';        // 宽字符常量，类型 wchar_t，表示中文字符 '中'
    
    printf("c1: %c, c2:%c, wc: %lc", c1, c2, wc);
	return 0;
}
```

# 字符串
- 字符串的初始化通常用strcpy_s， 这样更安全.
## 字符串的定义方式
```c
#include <stdio.h>

int main(void) {
	
	// --- A --- : 普通字符串
	// 方式一: 字符串数组
	char str[50] = "Hello World!";
	
	// 方式二: 指针字符串
	// 指针字符串定义后无法修改， 因此通常与const连用。因为修改指针字符串时会出现bug， 但是编辑器不报错
	const char* string = "Welcome!";
	puts(str);
	puts(string);
	printf("%s", str);
	
	// --- B --- : 宽字符串
	const wchar_t* wstr = L"世界";
	printf("%ls", wstr);
	return 0;
}
```
## 字符串的复制
- strcpy_s(目标字符串， 目标字符串大小， 被复制字符串)
- strncpy_s(目标字符串， 目标字符串大小， 被复制字符串， 最大复制长度)
- Note： 二者均返回一个errno_t类型的变量， 用于判断是否复制成功
```c
#include <stdio.h>
#include <string.h>

int main(void) {
	const char* str = "Hello World";
	char dest[50];
	char destN[50];
	
	errno_t result = strcpy_s(dest, sizeof(dest), str);
	if (result == 0) {
		puts(dest);
	}
	else {
		puts("Error occurred while copying string");
	}
	
	size_t max_copy = 5;
	errno_t resultN = strncpy_s(destN, sizeof(destN), str, max_copy);
	if (resultN == 0) {
		puts(destN);
	}
	else {
		puts("Error occurred while copying string");
	}


	return 0;
}
```
## 字符串拼接
- strcat_s(目标字符串， 目标字符串大小， 被拼接字符串)
- strncat_s(目标字符串， 目标字符串大小， 被拼接字符串， 最大拼接数)
- Note: 二者均返回一个int类型，为0时说明拼接成功
```c
#include <stdio.h>
#include <string.h>

int main(void) {

	const char* string = "Hello World";
	char dest[50];
	char destN[50];
	strcpy_s(dest, sizeof(dest), "Welcome");
	strcpy_s(destN, sizeof(dest), "Welcome");
	
	int result = strcat_s(dest, sizeof(dest), string);
	if (result == 0) {
		puts(dest);
	}
	else {
		puts("Error occurred while cating string");
	}
	
	int max_cat = 5;
	int resultN = strncat_s(destN, sizeof(destN), string, max_cat);
	if (resultN == 0) {
		puts(destN);
	}
	else {
		puts("Error occurred while cating string");
	}

	return 0;
}
```
## 字符串长度测量
- strlen
```c
#include <stdio.h>
#include <string.h>

int main(void) {

	char str[50] = "Hello World";
	size_t size_str = strlen(str);
	printf("%zd", size_str);

	return 0;
}
```
## 字符串输入
- gets_s(缓存字符， 缓存字符大小)。输入换行符后， 输入结束
```c
#include <stdio.h>
#include <string.h>

int main(void) {
	char buffer[50];

	if (gets_s(buffer, sizeof(buffer)) != NULL) {
		puts(buffer);
	}
	else {
		puts("Error Occurred");
	}
	
	return 0;
}
```

## 字符串输出
- sprintf_s(缓存字符， 缓存字符大小， 格式化输出字符串， 格式化变量若干)
```c
#include <stdio.h>
#include <string.h>

int main(void) {

	char buffer[50];
	int num1 = 10;
	int result = sprintf_s(buffer, sizeof(buffer), "num is %d", num1);
	if (result > 0) {
		puts(buffer);
	}
	else {
		puts("Error occurred");
	}

	return 0;
}
```
