| 维度       | 栈内存（Stack）                        | 堆内存（Heap）                                     |
| -------- | --------------------------------- | --------------------------------------------- |
| **分配方式** | 系统自动分配 / 释放（编译器控制）                | 程序员手动申请（`malloc`/`new`）、手动释放（`free`/`delete`） |
| **分配速度** | 极快（CPU 直接操作，无系统调用）                | 较慢（需系统调用，涉及内存块查找 / 管理）                        |
| **内存大小** | 固定且小（Linux 默认 8MB/Windows 默认 1MB） | 大（几乎占满剩余物理内存）                                 |
| **生命周期** | 随函数 / 作用域结束自动销毁（临时性）              | 直到手动释放或程序结束（持久性）                              |
| **分配规则** | 先进后出（FILO），连续内存块                  | 无序分配，内存块可能碎片化                                 |
| **访问效率** | 高（栈指针直接寻址）                        | 低（需通过指针间接寻址）                                  |
| **错误类型** | 栈溢出（Stack Overflow）               | 内存泄漏（Memory Leak）、堆溢出                         |
| **使用场景** | 局部变量、函数参数、返回值                     | 动态大小数据（数组 / 对象）、全局生命周期数据                      |
| **碎片问题** | 无（连续分配，自动回收）                      | 有（频繁申请 / 释放导致内存碎片）                            |
# 堆内存(heap memory)
- 分配： malloc
- 重分配： realloc
- 释放： free
```c
#include <stdio.h>
#include <stdlib.h>
#include <errno.h>
#include <string.h>

typedef struct {
	char* name;
	int taskcount;
} Project;

Project* create_project(char* name, int* taskcount);
void free_project(Project* project);

int main(void) {
	char* worker_name = "YnBn";
	int worker_tasks = 5;
	Project* project = create_project(worker_name, &worker_tasks);
	if (project == NULL) {
		perror("Failed to create project");
		return 1;
	}
	printf("Project Name: %s, Task Count: %d\n", project->name, project->taskcount);
	free_project(project);

	return 0;
}

Project* create_project(char* name, int* taskcount) {
	Project* project = (Project*)malloc(sizeof(Project));
	if (project == NULL) {
		perror("Error allocating memory for project");
		return NULL;
	}

	project->name = (char*)malloc(strlen(name)+1);
	if (project->name == NULL) {
		perror("Error allocating memory for project name");
		free(project);
		return NULL;
	}
	strcpy_s(project->name, strlen(name) + 1, name);

	project->taskcount = (int*)calloc(1, sizeof(int));
	if (project->taskcount == NULL) {
		perror("Error allocating memory for project taskcount");
		free(project);
		return NULL;
	}
	project->taskcount = *taskcount;
	return project;

}

void free_project(Project* project) {
	if (project != NULL) {
		free(project->name);
		free(project->taskcount);
		free(project);
	}
	return;
}
```
