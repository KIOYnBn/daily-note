# inttypes.h头文件用法
## PRI格式化输出格式
* 要点： 
	1. %要与双引号贴着
	2. 双引号与PRI有空格隔开
* 示例
```c
#include <stdio.h>
#include <inttypes.h>
#include <stdint.h>

int main(void) {
	
	int32_t num1 = 32;
	int32_t num2 = 33;
	printf("the sum of %" PRId32 " and %" PRId32 " is %" PRId32 "", num1, num2, num1+num2);

	return 0;
	
}
```
