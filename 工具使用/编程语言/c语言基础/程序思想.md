# 查表思维
* 内容： 
	1. 识别： 判断程序中可以用查表来替代的逻辑部分
	2. 设计： 创建包含所有可能输入和对应输出的数组
	3. 实现： 编写代码。使用输入作为索引或键来查询表中对应输出
	4. 
* 替代逻辑判断
	* 见micro_frank的c语言前篇p138
# 回调函数及应用
- 示例
```c
#include <stdio.h>
#include <stdlib.h>

typedef enum {
	ATTACK,
	DEFENSE,
} PlayerAction;

typedef void(*ActionFuncions)(char* name);

void attack(char* name) {
	printf("%s is attacking!\n", name);
}

void defense(char* name) {
	printf("%s is defending!\n", name);
}

ActionFuncions actionFunctions[] = {
	[ATTACK] = attack,
	[DEFENSE] = defense,
};

void performAction(PlayerAction action, char* name) {
	if (action >= ATTACK && action <= DEFENSE) {
		actionFunctions[action](name);
	} else {
		printf("Invalid action!\n");
	}
}

int main(void) {
	char playerName[50] = "YnBn";
	performAction(ATTACK, playerName);
	performAction(DEFENSE, playerName);

	return EXIT_SUCCESS;
}
```
