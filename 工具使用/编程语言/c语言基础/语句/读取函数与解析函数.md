# 读取函数
* fgets(赋值变量, 缓冲区大小， 输入流)
```c

#include<stdio.h>
int main(void) {
	char buffer[50];
	int num1;
	
	fgets(buffer, sizeof(buffer), stdin);
	sscanf_s(buffer, "%d", &num1);

	return 0;
}
```

# 解析函数
## scanf_s
* 从stdin解析内容
* 格式: 
	* scanf_s(解析格式， 赋值变量)
```c
#include <stdio.h>
int main(void) {
	int input_number;
	printf("请输入一个整数\n");
	scanf_s("%d", &input_number);
	printf("您输入的整数为%d\n", input_number);
	return 0;
}
```

## sscanf_s
* 作用： 从内存字符串中解析
* 格式: sscanf_s(解析字符串, 解析格式, 赋值变量)
```c
#include<stdio.h>

int main(void) {

	char buffer[50];
	int num1;
	fgets(buffer, sizeof(buffer), stdin);
	sscanf_s(buffer, "%d", &num1);

	return 0;
}
```
