# 构建模型
## 基本模型构建
### 构建块
```python
{
import torch.nn as nn

class MLP(nn.Module):

    def _init__(self, input_dim, hidden_dim, output_dim):

        super().__init__()

        self.hidden = nn.Linear(input_dim, hidden_dim)

        self.relu = nn.ReLU()

        self.output = nn.Linear(hidden_dim, output_dim)

    def forward(self, x):

        x = self.hidden(x)

        x = self.relu(x)

        x = self.output(x)

        return x
}
```

class mySequential(nn.Module):

        super().__init__()

        for block in args:

            self._modules[block] = block

    def forward(self, x):

        for block in self._modules.values():

            x = block(x)

        return x
}

## 权重衰退-weight-decay
* 含义： 如公式5中的$(1-\eta \lambda)w_t$ ， 在梯度传播之前， 将上次的梯度减小， 即衰退——$1-\eta \lambda <= 1$ 
* 作用： 通过限制参数值的选择范围控制模型容量， 进而避免过拟合
	* [[数据#模型容量与数据复杂度|参数值范围, 模型容量， 过拟合， 欠拟合之间的关系]]
### 数学原理
* 使用均方范数作为硬性限制
$$
\begin{align}
min \quad L(w, b) \tag{1} \\
subject \quad to \quad ||w||^2 <= 0 \tag{2} 
\end{align}
$$
---
* 使用均方范数作为柔性限制（常用）
$$
\begin{align}
min \quad l(w, b) + \frac{\lambda}{2}||w||^2 \tag{3}  \\
\frac{\partial}{\partial w}(l(w, b)+\frac{\lambda}{2}||W||^2) &= \frac{\partial l(w, b)}{\partial w} + \lambda w \tag{4} \\
w_{t+1} &= (1- \eta \lambda)w_t - \eta \frac{\partial l(w_t, b_t)}{\partial w_t} \tag{5}
\end{align}
$$
---
* 超参数： $\lambda$ , 数值通常选择$1^{-3}$ 等
### 代码实现
* 从头实现
```python
{
def l2_penalty(w):
	return torch.sum(w.pow(2)) / 2

l = loss(net(x), y) + lambda * l2_penalty(w)
}
```

## 丢弃法——Dropout
* 理念： 在层与层之间加入噪音
* 效果： 效果同正则化， 可以减小模型复杂度， 避免过拟合
* 应用场景： <span style="color:green">全连接层</span>的输出。只应用在<span style="color:green">训练时</span>
* 常见取值： 0.1， 0.5， 0.9
### 数学原理
$$
\begin{align}
x_i^{'} &= 
\begin{cases}
0, \quad with \; probality \;p \\
\frac{x_i}{1-p}, \quad otherise
\end{cases} \tag{6} \\
h &= \sigma(W_1 x+ b_1) \tag{7} \\
h^{'} &= drouput(h) \tag{8} \\
o &= W_2 h^{'} + b_2 \tag{9} \\
y &= softmax(o) \tag{10} 
\end{align}
$$
---
### 代码实现
* 从头实现
```python
{
def dropout_layer(x, dropout):
	assert 0 <= droput <= 1
	if dropout == 1:
		return torch.zeros_like(x)
	if dropout == 0:
		return x
	mask = (torch.randn(x.shape) > dropout).float()
	return mask * x / (1.0 - dropout)
}
```
