# 数据处理
## CSV文件
```python
{
import pandas as pd

data_file = "./data"
data = pd.read_csv(data_file)
inputs, outputs = data.iloc[:, :-1], data.iloc[:, -1]

# 将异常值替换为平均值
inputs["values"] = inputs["values"].fillna(inputs["values"].mean())

# 对字符串列进行独热编码
inputs = pd.get_dummies(inputs, dummy_na=True, dtype=int);

}
```

# 模型容量与数据复杂度

<p style="color: green; text-align: center;">  模型容量需要匹配数据复杂度， 否则会导致欠拟合和过拟合 </p> 
## 模型容量
* 定义： 拟合各种函数的能力
* 模型容量的影响： 
	* 低容量模型难以拟合训练数据， 过低容易产生欠拟合
	* 高容量模型可以记住训练数据， 过高容易产生过拟合
* 影响因素
	* 参数个数
	* 参数值的选择范围
---
## 数据复杂度
* 影响因素： 
	* 样本个数
	* 每个样本元素个数
	* 时间， 空间结构
	* 多样性
