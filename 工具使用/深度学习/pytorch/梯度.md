### 梯度下降
<p style="text-align: center"> 计算公式 </p>
$$w_t = w_{t-1} - \eta \frac{\partial \rho}{\partial w_{t-1}}$$
* 符号解释
	* $w_o$ 为初始值
	* $\eta$ ： 学习率， 
	* 步长： $\eta \frac{\partial \rho}{\partial w_{t-1}}$, 学习率和梯度乘积
	* 步长和学习率不能大也不能小
	* <img src="./pictures/chapter-梯度计算.png" width=300>


## 代码实现
```python
{
import torch

x = torch.arange(4.0)
x.reqires_grad_(true)

# 计算公式
y = 2 * torch.dot(x, x)

# 反向累积
y.backward()

# x.grad == 4 * x

# 梯度清零
x.grad.zero_()
}
```

# 梯度爆炸或消失问题
## 原因
* 对于d层的神经网络的$w_t$ 会出现d-t次矩阵乘法
$$ 
\begin{align}
\frac{\partial l}{\partial w^t} &= 
\frac{\partial l}{\partial h^d}
\frac{\partial h_d}{\partial h^{d-1}}
....
\frac{\partial h{t+1}}{\partial h^t}
\frac{\partial h_t}{\partial w^t}
\end{align}
$$
---
* 示例
$$
\begin{align}
1.5^{100} &\approx 4*10^{17} \\
0.8^{100} &\approx 2*10^{-10}
\end{align}
$$
---
* 使用sigmoid容易引起
	* [[大模型发展历史#^9cd2dc|sigmoid公式]]
## 影响
### 梯度爆炸
* 值超出值域（Infinity)
	* 对于16位浮点数尤为严重
* 对学习率敏感
	* 学习率太大——大参数——更大的梯度
	* 学习率太小——训练无进展
	* 可能需要在训练过程中不断调整学习率
### 梯度消失
* 梯度值变成0
	* 对16位浮点数尤为严重
* 训练没有进展， 无论如何选择学习率
* 仅顶部层训练较好， 无法让神经网络更深
## 解决办法
* 目标： 让梯度值位于合理范围内
* 方法
	* 将乘法变加法
		* ResNet, LSTM
	* 归一化
		* 梯度归一化， 梯度裁剪
	* 合理的权重初始和激活函数
#### 合理的权重初始
* 设置成具有方差固定， 均值为0的， 如正态分布
* [[常见的参数初始化方法.pdf|常见的权重初始化方法]]
	* 全零初始化
	* 随机初始化（均匀分布， 正态分布）
	* Xavier初始化： 适合sigmoid和tanh激活函数
	* Kaiming初始化（He初始化）： 适用于relu
	* 偏置通常设置为0.
### 合理的激活函数
* 方法： 让每层的输出和梯度看作随机变量， 让其均值和方差保持一致
* 结论： 一般选择relu即可。sigmoid容易引起梯度消失
