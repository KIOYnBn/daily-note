## 普通前馈网络
FFN: FeedForward Network
### 1. 数学原理
普通双层前馈网络结构为：**线性升维 → 激活 → 线性降维**

$$\begin{align*}
&H = \text{ReLU}(X W_1 + b_1) \tag{1}\\ 
&Y = H W_2 + b_2 \tag{2} \\
&X\in\mathbb{R}^{B\times D}:单样本二维特征（无序列维度） \\
&W_1\in\mathbb{R}^{D\times pf\_dim}: 升维权重 \\
&W_2\in\mathbb{R}^{pf\_dim\times D}:降维权重
\end{align*}$$
**核心数学特性**：运算基于**整行特征全局映射**，权重与“位置”无关，只对特征维度做全局变换。
### 2. 代码实现
```python
import torch.nn as nn
import torch.nn.functional as F
class NormalFFN(nn.Module):
    def __init__(self, hid_dim, pf_dim, dropout):
        super().__init__()
        # 标准全连接前馈
        self.fc1 = nn.Linear(hid_dim, pf_dim)
        self.fc2 = nn.Linear(pf_dim, hid_dim)
        self.dropout = nn.Dropout(dropout)

    def forward(self, x):
        # x: [B, L, D]
        x = self.dropout(F.relu(self.fc1(x)))
        x = self.fc2(x)
        return x
```

## 位置前馈网络
Positoin-wise FeedForward Network:PWFFN
*特点*： linear换成conv1d(1x1卷积核）。每个特征都是单独的变化的， 彼此之间没有关联。
```python
class PositionwiseFeedforward(nn.Module):
    """Position-wise feedforward network."""
    def __init__(self, hid_dim: int, pf_dim: int, dropout: float):
        super().__init__()
        self.fc_1 = nn.Conv1d(hid_dim, pf_dim, 1)
        self.fc_2 = nn.Conv1d(pf_dim, hid_dim, 1)
        self.do = nn.Dropout(dropout)
    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """Forward pass of feedforward network."""
        x = x.permute(0, 2, 1)
        x = self.do(F.relu(self.fc_1(x)))
        x = self.fc_2(x)
        x = x.permute(0, 2, 1)
        return x
```
