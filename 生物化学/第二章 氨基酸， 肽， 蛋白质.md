# 氨基酸
## 蛋白质水解
- 水解方式： 酸， 碱或 蛋白水解酶
	- 酸水解
		- 缺点： [[知识块#酸水解时色氨酸被破坏的原因|色氨酸被破坏]]， [[知识块#酸水解时羟基氨基酸部分水解|羟基氨基酸（丝氨酸， 苏氨酸）部分水解]]， 天冬酰胺和谷氨酰胺的酰胺基被水解
		- 优点： 仅产生L-氨基酸
	- 碱水解缺点： 
		- 缺点： [[知识块#蛋白质碱水解会引起消旋的原因|消旋]]；[[知识块#碱水解时， 精氨酸脱氨|精氨酸脱氨]]
	- 酶水解： 
		- 缺点： 水解时间长， 需要多种水解酶协同作用才能完全水解
- 水解类型
	- 完全水解： 产物为氨基酸
	- 部分水解： 产物为肽段和氨基酸
## 氨基酸结构
- 蛋白质中氨基酸都是*L型*
- 除胱氨酸和酪氨酸外， 其他氨基酸都能溶于水

- 当前发现氨基酸约300种， 其中20种为基本氨基酸， 少部分为非常见蛋白质氨基酸。余二百多种为非蛋白质氨基酸

# 蛋白质的酸碱性
## 氨基酸等电点
- 氨基酸醋鱼静电荷为0时的PH。缩写为*pI*。
- 等电点计算： 等点兼性离子两边的$pK_a$的平均值
	- 对于侧链不含可解离基团的氨基酸： 直接pK1+pK2
	- 对于侧链含可解离基团的氨基酸： 判断哪个状态是兼性氨基酸， 取就近两个pK的平均值即可

# 氨基酸的旋光性和光谱性质
- 参与蛋白质组成的氨基酸在在*红外区和远紫光区*具有光吸收能力。*芳香族*氨基酸在*近紫光区*具有光吸收能力。
- 分光光度法定量分析的依据*Lambert-Beer定律*：
	- $A = lg \frac{I_0}{I} = -lg T = \varepsilon cl$
		- A: 吸光度
		- $I_0$: 入射光强度
		- I: 透射光强度
		- T: 透光率
		- $\varepsilon$： 摩尔吸收系数
		- c： 浓度
		- L： 光程厚度
- 核磁共振（NMR）： *外磁场*下*原子核*吸收射频能量的波谱技术

# 氨基酸混合物的分析与分离
## 层析
### 分配层析
- 层析， 又名*色谱*， 由*固定相*和*流动相*组成。
- *分配系数*： 混合物在两相之间的分配情况。
	- *分配定律*： 一定温度下*平衡后*， 溶质在两相之间的*浓度比值*极为分配系数。
		- $K_d = \frac{C_M}{C_S}$
			- M: 流动相
			- S： 固定相
	- *有效分配系数*： $K_{eff} = \frac{物质在M相中的总量}{物质在S相中的总量}$.
		- 示例： 液相-液相——$K_{eff} = \frac{C_M V_M}{C_S V_S} = K_d R_v$
- 物质分配除了*液相-液相*， 还有*液相-固相*， *液相-气相*等
- 层析法分离氨基酸， 取决于氨基酸之间的*分配系数差异*
- 逆流分配： 
	- 转移n次后， k号管的物质含量： $T_{n, k} =  \frac{n! · p^{n-k+1} · q^{k-1}}{(n-k+1)!·(k-1)!}$
		- p： 固定相中的总量比例；q： 流动相中的总量比例
	- *分布系数大的移动速度快*
#### 类型
##### 柱层析
- 收集的组分用*茚三酮*显色， 以氨基酸量对洗出的液体体积作图， 可得*洗脱曲线*， 每个*峰*为一种氨基酸。
![[柱层析.png|200]]

##### 纸层析
- *固定相*： 滤纸吸附的水
- *流动相*： 溶剂系统
- 操作： 
	1. 层析时， 将混合物点在滤纸的边角上（原点）
	2. 用溶剂系统在一个方向上进行展开
	3. （双向纸层析）烘干滤纸， 旋转90°， 用另一种溶剂系统在另一个方向进行层析， 得到*双向纸层析谱*
	4. （如果混合物中氨基酸类型少， 分配系数差异大， 用（单向纸层析即可）
- 比移值： 溶质峰值所处的位置（类似于管号）
	- $R_f = \frac{x}{x+y} = \frac{K_d V_M}{K_d V_M + V_S}$
##### 薄层层析
- *优点*： 分辨率高， 样品量少， 速度快， 支持剂种类多
- *步骤*
	1. 将支持剂在玻璃板上涂成均匀薄层
	2. 将样品滴在薄层一段
	3. 添加溶剂系统进行展开
![[薄层层析.png|300]]
### 离子交换层析
- 基于*氨基酸电荷*的层析方法
- *填充物*： 离子交换树脂（水不溶性高分子）， 通常为球形
- 氨基酸分离常用*强酸性阳离子交换树脂*。
	- 树脂中的酸性基团解离出$H^+$， 与带正电的氨基酸发生交换， 使其结合在树脂上
- *强酸性阳离子交换树脂*： 
	1. 用碱处理树脂成*钠型*
	2. 酸性氨基酸混合液上柱， 氨基酸带正电， 与树脂发生交换， 结合在树脂上。
	- 与树脂的吸引力大小： 碱性氨基酸> 中性氨基酸 > 酸性氨基酸。因此洗脱顺序为： 酸性氨基酸 > 中性氨基酸 > 碱性氨基酸
	- 仪器： *氨基酸分析仪*

# 肽
## 肽与肽的结构
- *常见的概念*：
	- 寡肽： 通常指含几个或十几个氨基酸残基的肽
	- 多肽： 通常指多余二十个， 小于100个残基的肽
	- 氨基酸残基： 肽链中的氨基酸
	- 氨基端： 多肽链的主链中含有α氨基的末端氨基酸
	- 羧基端： 多肽链的主链中含有α羧基的末端氨基酸
	- 环状肽： 肽链的两个游离的末端基团连接起来形成环状肽。
- 氨基酸结构的书写: 通常将N端写在左边
- *肽结构*：
	- 肽键是一个*共振杂化体*： 酰胺键上N的孤电子对与羧基O之间发生共振相互作用。
		- 因此C-N键具有双键性质， 使肽键成为一个平面（*肽平面*）—$C_\alpha, O, C-N, C_\alpha, H$位于同一平面
![[肽键的共振杂化体.png|500]] ^1da6c0
## 肽的物理和化学性质
### 酸碱性
在pH0~14范围内， 肽键的酰胺氢不解离， 短肽的酸碱性质取决于游离的末端$\alpha-NH_2, \alpha-COOH$以及侧链可解离官能团。
[[第一章 生命分子基础#^fff7ae|Henderson-Hasselbalch方程]]
### 化学反应
*双缩尿反应Biuret Reaction*: 含有两个及以上的肽键的化合物与$CuSO_4$碱性溶液反应， 生成*紫红色或紫色复合物*。
## 天然活性肽
- 自然界中活性肽的形成途径： *核糖体途径*， *细菌或低等真菌产生的肽类抗生素*
	- 核糖体途径的氨基酸几乎都为*L型*
- 常见的天然活性肽：
	- 还原型谷胱甘肽GSH： 维持半胱氨酸巯基处于还原态。
	- 放线菌素D： 结构中有D型缬氨酸， 与DNA结合抑制转录
	- α-鹅膏蕈碱： 环状八肽， 与真核生物RNA聚合酶Ⅱ结合抑制RNA合成。

### 质谱法MS
- 通常用串联质谱MS/MS
#### ESI-MS/MS： 电喷雾离子化
1. 酸性溶剂加蛋白作为样品（液相）
2. 通过高压电场， 使液体分散为小液滴（静电斥力大于表面张力）
3. 小液滴在“飞行”过程中逐渐蒸发， 到达MS时， 几乎只剩下带电蛋白质（肽段）
4. 带电蛋白质（肽段）根据质荷比分离。
5. 带点蛋白质进入*碰撞池*（惰性气体分子撞击蛋白质/肽段， 使其随机肽键断裂， 通常只断裂一个肽键）， 到达第二个MS

# 氨基酸序列与生物进化
- 概念
	- 序列同源性： 两个基因具有明显的序列相似性。
	- 同源蛋白质： 具有序列相似性的基因编码的蛋白质
		- 直向同源蛋白质： 同源蛋白质来自不同物种，通常结构和功能相似。
		- 共生同源蛋白质： 同源蛋白质来自不同物种， 通常结构相似。
	- 蛋白质家族： 具有特定的结构和功能特点。
	- 结构域： 蛋白质家族中较长的相似序列
	- 序列模体motif： 具有家族标志性的不太长的肽段
	- 不变残基： 功能必需的氨基酸， 进化过程中保守的
	- 可变残基： 进化过程中不保守的
- 一般来说， 进化位置上越远， 氨基酸序列的残基差异数越多
	- 细胞色素c的氨基酸序列通常用来分析物种间分类关系和绘制系统进化树。
# 肽和蛋白质的合成
- 肽获得途径： 动植物组织中分离；基因工程；化学合成
## 化学合成
保护基要求： 接肽时起保护作用；接肽后容易去除；不会断裂肽键
氨基保护基： BOC叔丁氧羰基；Fmoc芴甲氧羰基
	二者用$CF_3COOH$去除
羧基保护基： 用盐或酯。如钠盐， 钾盐， 甲酯，乙酯，叔丁酯
	甲酯， 乙酯用皂化去除， 但容易消旋
	叔丁酯用酸去除
缩合剂：*DCC*——直接与被保护的氨基， 羧基反应， 成肽
	DCC： $N,N^{'}-二环己基二亚胺$。反应后成固体， 过滤去除
*固相肽合成SPPS(solid-phase peptide synthesis)*: C末端氨基酸与氯甲基聚丙乙烯-二乙烯苯树脂反应。DCC为缩合剂。最后用$CF_3COOH+HF$使肽和树脂脱离， 并切除保护基。——*固相肽合成仪*
