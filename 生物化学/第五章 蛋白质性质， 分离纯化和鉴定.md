# 蛋白质性质
## 溶解性
1. *中性盐*可以改变蛋白质等电点和滴定曲线：通过与蛋白质结合， 改变其电荷。
2. *等离子点*：蛋白质质子供体解离出的质子数与质体受体基团结合的质子数相等时的*pH*.
## 胶体
1. 蛋白质溶液属于*胶体*。大多数蛋白质大小处于1-10nm间， 除了特别小的肽段
2. 由于蛋白质溶液的分散相质点是蛋白质本身， 其也是*均相系统*。一般只有*真溶液*才是均相系统
3. 分散相质点在胶体中保持稳定的条件：
	1. 大小在1-100nm之间， 介质对这种质点碰撞合力不为零， 使其作*布朗运动*。
	2. 质点具有同种符号电荷。
	3. 质点与水形成*水化层*。

# 蛋白质沉淀（结晶）
- 原理： 1）*破坏水化层*。2）使其静电荷为0
- 方法
	1. 盐析法：中性盐， 破坏水化层
	2. 有机溶剂沉淀法：极性有机溶剂， 破坏水化层， 降低介电常数， 引起*变性*。
	3. 重金属盐沉淀法：pH>等电点时， 与重金属离子结合形成不溶性盐， 引起*变性*。
	4. 生物碱/某些酸沉淀：形成不溶性盐
	5. 加热变性：构象变化， 疏水基外露， 破坏水化层。

# 蛋白质分离纯化步骤
## 1. 前处理
- 目的：将蛋白质从组织/细胞中释放
1. 破碎细胞
	- 动物：
		1. 剔除结缔组织， 脂肪
		2. 捣碎机/匀浆器/超声处理破碎细胞
	- 植物：
		1. 种子要去皮
		2. 研磨/纤维素酶
	- 细菌：超声震荡/溶菌酶
2. 差速离心分离
## 2. 粗分离
1. 粗分离
	- 方法1：[[#二. 选择性沉淀|选择性沉淀]].包括盐析， 等电点沉淀， 有机溶剂分离
	- 方法2：不能沉淀的， 用*层析法*。
- 换液：缓冲液重溶
- [[#三. 脱盐方法|脱盐]]：透析/超滤
## 3. 细分离
柱层析。
## 4. 结晶
- 通过控制*温度， 盐析， 有机溶剂， 调节pH*， 使溶液到达过饱和。
- Note：变性的蛋白不能结晶。

# 蛋白质分离纯化的方法
## 一. 分离纯化的依据
1. 根据蛋白质溶解度
	- 影响溶解度的因素：
		1. pH
		2. 离子强度
		3. 介电常数
		4. 温度
2. 根据分子大小:主要为*凝胶过滤*。
3. 电荷：电泳， 离子交换层析
4. 结合性质：疏水相互作用层， SDS-PAGE， 亲和层析
## 二. 选择性沉淀
### 等电点沉淀
原理：
1. 等电点时， 蛋白质分子之间没有静电排斥和聚集沉淀
2. 利用蛋白质等电点时溶解度最低， 可以分离蛋白质混合物
### 盐析
1. *盐溶*： 低浓度时， 中性盐可以增加蛋白质溶解度
	原理： 蛋白质分子吸附某种盐类例子后， 发生静电排斥， 而与水的相互作用增强
2. *盐析*：*大量*中性盐的加入使水的活度降低， 溶液中大部分自由水变成盐离子的水化水。原先与水结合的蛋白质疏水基团暴露， 通过*疏水相互作用*沉淀
中性盐影响蛋白质溶解度的能力可以通过离子强度表达
$$I = \frac{1}{2}\Sigma C_iZ_i^2$$
示例：2mol/L的硫酸铵：$\frac{(4*1)+(2*4)}{2}=6$
### 有溶剂分级分离
原理
	1. 改变介电常数。根据库仑定律， 介电常数的降低将增加电荷吸引力
	2. 破坏水化层。水溶性非离子聚合物， 如聚乙二醇， 可以脱去蛋白质水花层。
## 三. 脱盐方法
### 透析
利用蛋白质分子不能通过*半透膜*。， 分离蛋白质与小分子物质
### 超滤
通过压力或离心力强行使水和其他小分子通过*半透膜*， 实现*浓缩*和*脱盐*的目的。
## 四. 密度梯度超速离心
*介质密度梯度*
每种蛋白质颗粒沉降到与其自身密度相同的梯度
常用的密度梯度有：蔗糖梯度， 聚蔗糖梯度

# 蛋白质相对分子质量测定
## 凝胶过滤
*原理*
	1. $lg_{10}M_r = a-bV_e$
		$a，b为常量， M_r为相对分子质量， V_e为洗脱体积$
	2. 通过标准蛋白已知的$M_r$和$V_e$绘制标准曲线
	3. 依据$V_e$找出$M_r$
缺点：*仅限于球蛋白*
优点：*待测样品可以不纯*。
## SDS-PAGE
SDS：十二烷基硫酸钠
	1. 一种*变性试剂*， 能破坏蛋白质分子中氢键和疏水相互作用， 使肽链展开
	2. 自身的负电荷能掩盖蛋白本身的电荷
*考马斯亮蓝*染色
$lgM_r = a -b\mu_{R}$
缺点：迁移率收到亲水基团和疏水基团比例影响， 误差较大
## 沉降速度法
原理:
$$\begin{align*}
&M_r = \frac{RTs}{D(1-\overline{v}e)} \tag{1} \\
&\frac{dlg\ x}{dt}=\frac{sw^2}{2.303} \tag{2}
\end{align*}$$
	R:气体常数；T:绝对温度；$\overline{v}$:偏微比容， 为0.74；$\rho$：溶剂密度；D：扩散系数（单位时间面积通过的蛋白质质量）；s：沉降系数；x：旋转中心到界面的径向距离
工具：超速离心机
通过界面处不同浓度的折射率不同， 观察界面移动。
## 质谱法
原理：$\frac{m}{z} = \frac{M+nH}{n}$
	M:蛋白质分子质量。
	通过任意两个相邻的峰可以确定M
常用的为*电喷雾离子化质谱*。
