# 与事实相反
## 与现在事实相反
```
if 主语 动词过去式， 主语 would/could/might 动词原形
```
示例： `If I were you, I would accept the offer`

## 与过去事实相反
```
If 主语 had 过去分词, 主语 would/could/might have 过去分词
```
* There are targets that would have been nice to hit

## 与将来事实相反
```
If 主语 were to/should 动词原型, 主语 would/could/might 动词原形
```
* If the sun were to rise in the west, I would not change my mind

# 愿望
## 现在的愿望——与现在事实相反
```
wish 主语 动词过去式
```
* I wish I were taller

## 过去的愿望——与过去相反
```
wish 主语 had 过去分词
```
* I wish I had studied harder for the exam

## 将来的愿望
```
wish 主语 would 动词原形
```
* I wish it would stop raining

# 表示建议， 要求， 命令
```
主语 建议类动词 that 主语 (should) 动词原形
```
* The teacher suggested that we read more books
* 建议类动词： insist, suggest, recommend, order, demand, propose

# as if/though引导的状语从句
* 如果从句的描述与现实相反或可能性很小， 用虚拟语气
	* 现在： 过去式
	* 过去： had 过去分词
