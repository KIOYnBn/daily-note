# 理念
综述的理念：
	1. 综： 筛选， 分类领域的文献
	2. 述： 评价， 论证领域的工作
## 科学论证的基本结构
1. 观点
2. 解释
3. 证据
4. 收尾：总结， 推论， 过渡到下一个内容

---
# 类型
## 1. 概览型
*场景*： 初学者入门， 课程报告
目标： 描述已有的研究内容
关键词： “X研究了..", "Y发现了”
*AI写的论文大多都是这种*。
## 2. 评析型
场景： 批判性思维训练
目标： 比较评析已有的研究
关键字： 有哪些争议
## 3. 论证型
场景： 论文综述， 开题报告， 基因申请
目标： 论证当前研究逻辑
关键词： “已有研究尚未解决..， 因此本文聚焦.."
## 4. 理论构建型
场景： 高水平综述期刊
目标： 提出新理论
关键词： ”基于上述发现， 本文提出一个整合模型“

# 错误范例
## 1. 学者点名型
*既没有筛选分类， 又没有评价论证*
```
学者A认为...
学者B发现...
学者C研究了...
综上所述， 本文在研究基础上...
```
## 2. 教科书式
缺点：
	只有整理， 没有*整合*。
	重复性高， 查重过不了
```
X的定义是..
X的测量方法有..
X的相关理论有..

综上所述， 本文将在前任研究的基础上..
```

# 论证型综述
## 思路 
1. Findings： 提炼已有发现的主要观点， 发现
2. Gains： 争议， 盲点， 新方法， 新思路
3. Connections： 切入策略， 填补空白
## 目标
### 1. 研究定位：重要， 创新
*定位到研究领域的同一， 或上一层的主题层级*。
```
过去研究主要集中在..
却忽略了一个重要的对象/情景/变量..
因此， 本研究聚焦于...， 尝试填补..的空白
```
### 2. 研究的构建逻辑：合理
```
先前研究显示， ...
这一发现启发我们, ...可能通过...基质影响....
因此， 本研究..., 并假设...
```
### 3. 实施方法： 可行， 优越
*根据相似度取舍*：借鉴其他领域相近的方法
```
多项研究采用...方法， 成功测量到...
这表明该方法适用于处理...类型的问题， 为本研究的设计提供了参考
因此， 本研究将采用..方法
```

# 文献的筛选， 描述， 组织
## 引用文献的筛选
*核心： 研究领域的主题层级*。
1. 研究质量的评估
2. 采用同一主题或上一层主题的文献
## 文献的描述
### 错误1：过于详细
应当着重于与本研究相关的部分
### 错误2： 过于泛化
1. 笼统介绍了谁做了什么：没有细节， 看不懂
2. 把*观点=证据*。作者提到和作者论文是两回事
3. *用泛化结论替代具体结论*：埋没了创新性
## 多文献组织
根据*研究之间的关系， 研究在论证中的作用*。
### 文献之间的关系
#### 1. 观点一致
分类多侧面论证
	不同的理论， 实证
	不同方法
	不同样本
![[文献组织示例.jpg|500]]
#### 2. 不一致
1. 不一致和冲突点
2. 如果倾向某一方， 分析二者的差别以及另一方的证据为何无效
