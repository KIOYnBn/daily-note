- HPLC：High Performance Liquid Chromatography，高效液相色谱
- 是以液体为流动相，*高压泵*驱动，样品流经*固体固定相色谱柱*，依据组分分配/吸附差异实现分离的分析技术。
- 适用：大分子、热不稳定、不易挥发、极性强物质（蛋白质、多肽、核酸、药物、代谢物、天然产物）。
## 原理
1. 样品溶解在液相体系，由高压泵推送；
2. 混合组分随流动相穿过色谱柱（固定相）；
3. 不同物质与固定相、流动相的作用力强弱不同：- 结合固定相越强 → 保留时间越长、出峰越晚
- 结合流动相越强 → 洗脱快、出峰早
4. 组分依次分离，经检测器转化为电信号，生成色谱图；
5. 通过保留时间定性、峰面积/峰高定量。
## 仪器核心组件
1. 流动相储液瓶: 提供洗脱溶剂，需脱气、过滤，防气泡与堵柱。
2. 高压恒流泵: HPLC关键：高压（40~400 bar），稳定流速，保证高效分离。
3. 进样器: 手动进样阀 / 自动进样器，精准定量上样。
4. 色谱柱（核心分离部位）: 填充超细多孔填料，决定分离效果；柱温箱控温，减少保留时间漂移。
5. 检测器 + 数据处理系统
	常见：UV紫外、DAD二极管阵列、荧光、示差折光、质谱联用（LC-MS）。
## HPLC 四大分离模式（按固定相分类）
### 反相色谱 RP-HPLC（最常用，占80%）
- 固定相：非极性（C18、C8）
- 流动相：极性（水+甲醇/乙腈）
- 分离规律：极性大 → 先洗脱；非极性/疏水强 → 保留久
- 适用：多肽、小分子药物、有机酸、黄酮
	- 蛋白/多肽分离几乎全用 RP-HPLC
### 正相色谱 NP-HPLC
- 固定相：极性（硅胶、氨基柱）
- 流动相：非极性有机溶剂
- 适用：异构体、脂类、弱极性物质
### 离子交换色谱 IEC
- 固定相：带电荷树脂
- 依据：分子电荷差异分离
- 适用：蛋白质、氨基酸、核酸、带电代谢物
### 凝胶过滤 / 体积排阻色谱 SEC
- 依据：分子大小分离
- 大分子不能进入填料孔 → 先洗脱
- 小分子钻入孔洞 → 保留时间长
- 适用：蛋白分子量测定、复合物分析
## 关键色谱参数
1. 保留时间 t_R: 组分出峰时间，用于定性。
2. 峰面积 / 峰高: 与物质浓度成正比，用于定量。
3. 分离度 R: 两相邻峰分开程度，R>1.5 视为完全分离。
4. 理论塔板数 N: 评价色谱柱柱效，数值越大、峰越窄、分离越好。
5. 拖尾因子: 评价峰形对称性，反映柱损伤、过载、条件不合适。

## 洗脱方式
1. 等度洗脱: 流动相比例全程不变，适合组分简单样品。
2. 梯度洗脱: 逐步改变流动相极性/比例，适合复杂混合物（多肽、蛋白水解液），缩短分析时间、提高分离度。
## HPLC 优缺点
### 优点 
1. 分离效率高、分辨率强
2. 常温分离，适合热不稳定生物大分子
3. 灵敏度高、定量准确、重复性好
4. 样品无需气化，适用范围极广
### 缺点
1. 仪器昂贵、耗材（色谱柱）成本高
2. 流动相需高纯试剂，维护要求高
## 应用
1. 多肽、蛋白质分离纯化
2. 酶解肽段指纹图谱（质谱前预处理）
3. 氨基酸、核苷酸、生物小分子定量
4. 结合LC-MS，实现蛋白组学定性+定量
5. 天然活性物质、代谢组样品分析
