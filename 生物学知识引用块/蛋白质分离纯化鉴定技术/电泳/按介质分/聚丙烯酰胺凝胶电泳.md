1. APS为催化剂
2. 加速剂： TEMED
3. 原理：TEMED催化APS产生自由基， 引发`丙烯酰胺`单体聚合， `甲叉双丙烯酰胺`与丙烯酰胺链间发生交联， 形成`三维网状` 

聚丙烯酰胺凝胶电泳（PAGE）
## 一、核心定义
- PAGE = Polyacrylamide Gel Electrophoresis
- 以聚丙烯酰胺凝胶作为支持介质与分子筛的电泳技术，依靠电荷差异 + 分子大小/构象筛分分离物质，
- 特点：超高分辨率、分离小分子片段
## 二、凝胶合成原理
- 单体：丙烯酰胺（Acr）
- 交联剂：甲叉双丙烯酰胺（Bis）
- 引发催化体系：
	1. APS（过硫酸铵）：提供自由基
	2. TEMED：加速自由基生成
- 聚合反应： Acr 在自由基作用下线性聚合，Bis 横向交联，形成三维网状多孔凝胶结构。
- 公式：$凝胶孔径大小 ∝ \boldsymbol{\dfrac{1}{Acr+Bis总浓度}}$
	- 胶浓度越高 → 网孔越小 → 适合分离小分子
	- 胶浓度越低 → 网孔越大 → 适合分离大分子
## 三、分类
1. 非变性PAGE（Native-PAGE）
	- 不加变性剂（无SDS、无尿素）
	- 蛋白/核酸保持天然空间构象、高级结构
	- 分离依据：分子电荷 + 大小 + 形状
	- 用途：蛋白寡聚体、酶活性检测、核酸二级结构、蛋白互作
2. 变性PAGE
	- （1）SDS-PAGE（蛋白专用）
		- 加 SDS 十二烷基硫酸钠
		- SDS使蛋白变性：解折叠为线性
		- 均匀包裹负电荷，掩盖蛋白本身电荷差异
		- 分离只看：蛋白质分子量大小
	- （2）尿素变性PAGE（核酸专用）
		- 高浓度尿素破坏氢键，核酸完全解链为单链
		- 用于：短片段单链DNA、RNA、SSR、突变检测
## 四、PAGE 为什么比琼脂糖凝胶分辨率高？
1. 琼脂糖凝胶孔径大，适合大片段核酸（几百 bp～Mb级）
2. PAGE 网孔致密均匀，小孔分子筛效应极强
3. 可完美区分：长度只差1~2 bp的短DNA片段
	- 经典应用：SSR微卫星、SSCP突变、短肽、小RNA
五、核酸PAGE 操作
1. 样本：短片段DNA（<500bp）、引物、小片段酶切产物
2. 电泳条件：高压、恒压运行
3. 显色方式（重点）：- 不能用EB/荧光染料染色（结合弱、背景高、看不清）
- 标配：银染显色（高灵敏、条带清晰）
4. 对比记忆- 大片段DNA → 琼脂糖凝胶 + EB/新型荧光染料
- 小片段/差异极小DNA → 尿素变性PAGE + 银染
 
 
 
六、SDS-PAGE 分离蛋白核心逻辑
 
1. 煮沸变性+SDS结合 → 蛋白全部变为线性、等电荷密度
2. 电泳时只受分子筛阻力：- 小分子蛋白：穿过网孔快，迁移距离远
- 大分子蛋白：阻力大，迁移慢
3. 染色：考马斯亮蓝 / 银染（微量蛋白）
4. 用途：蛋白分子量测定、蛋白纯度鉴定
 
 
 
七、PAGE 完整用途汇总
 
1. 核酸方向
 
- 短片段DNA分离、SSR/STR基因分型
- SSCP单链构象多态性（基因突变筛查）
- 小RNA、引物二聚体、寡核苷酸分离
 
2. 蛋白方向
 
- 蛋白分子量测定、纯度分析
- 天然蛋白复合体、同工酶分析
 
3. 补充：
早期人工非荧光版测序胶，也是高浓度尿素PAGE，配合放射自显影读条带
 
 
 
八、易混考点总结（极简背诵）
 
1. 介质：聚丙烯酰胺，Acr+Bis交联聚合
2. 核心作用：分子筛效应
3. 分辨率：远高于琼脂糖，适合小分子
4. 核酸PAGE显色：银染
5. 蛋白变性PAGE：SDS消除电荷，只按分子量分离
6. 天然PAGE：保留空间结构，看电荷+形态+大小
