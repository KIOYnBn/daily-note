## 线粒体
- 形态： 颗粒状或短棒状的囊状细胞器
- 结构： 
	- 两层膜。内膜形成嵴
	- 基质中具有独立遗传系统
- 作用： 化学能转化为ATP
## 质体
### 叶绿体
- 结构： 
	- 类囊体： 
		- 类囊体膜中含有转化所需的色素和电子传递系统
- 作用： 光能转化成化学能
## 内质网
- 膜质囊腔和小管彼此相通的膜系统
- 类型
	- 光面内质网： SER——脂质合成
	- 粗面内质网： RER——蛋白质合成
## 高尔基体
- 单层囊和泡组成
- 功能： 蛋白质分类加工包装运输
## 溶酶体
- 单层膜小泡， 起源于高尔基体
- 特点：内含多种水解酶
- 功能： 消化
## 液泡
- 植物细胞特有
- 特点： 体积大
- 功能： 运输， 存储物质
## 微体
- 单层膜小泡， 类型多样
- 动植物都存在
- 常见类型： 过氧化物酶体， 乙醛酸循环体
- 功能： 参与生化反应

## 细胞器比较
### 微体与液泡
| 项目    | 微体（Microbody）   | 液泡（Vacuole）        |
| ----- | --------------- | ------------------ |
| 大小    | 很小，微米级小泡        | 很大，成熟植物细胞中占主体      |
| 数量    | 细胞中**多个、分散**    | 植物细胞通常**一个大中央液泡**  |
| 主要功能  | **代谢反应**（氧化、解毒） | **储存、渗透压、维持坚挺**    |
| 内含物   | 酶类（氧化酶、过氧化氢酶等）  | 细胞液：水、糖、盐、色素、废物等   |
| 标志性酶  | 过氧化氢酶           | 无特定标志性酶            |
| 存在    | 动、植物、真菌细胞都有     | 主要在**植物细胞**；动物有小液泡 |
| 与代谢关系 | 活跃参与生化反应        | 多为储存，不直接参与主要代谢通路   |
