# 睾丸
* 睾丸功能
	* 产生精子
	* 分泌雄激素
* 来源
	* 生殖嵴——雄性生殖腺（睾丸）——精子发生场所
## 曲细精管（生精小管）
* 生精小管：睾丸实质部分， 周围是结缔组织薄层（弹性纤维和平滑肌细胞组成）管壁从内到外分别为支持细胞和生精细胞
* 生精上皮： 生精细胞+支持细胞
* 结构
	* 基底膜： 肌样细胞+成纤维细胞+胶原纤维
	- 基底间： 精原细胞， 细线期前的精母细胞
	 - 近腔间：细线期后的初级精母细胞， 次级精母细胞， 精细胞， 精子
## 细胞类型
* 支持细胞
	* 不规则型， 明显核仁， 与基底膜相连， 顶端直达管腔
	* 细胞质含有丰富内质网， 溶酶体， 脂肪滴， 致密小体， 高尔基体
		* 致密小体： 这里指生殖细胞的残余体， 在支持细胞中吸收
	* 作用
		* 提供营养
		* 合适的微环境
		* 参与血睾屏障
		* 对生精细胞起支持作用
		* 合成分泌雄激素合成蛋白
		* SSC： 精原干细胞
		* 管周肌样细胞：
* 间质细胞： leydig细胞
	* 形状： 体积大， 圆形/多边形， 胞质嗜酸性
	* 位于生精小管间
	* 功能
		* 分泌雄激素（睾酮）——免疫抑制——抑制支持细胞， 管周肌细胞， leydig细胞表达免疫调节因子——免疫豁免环境
			* 睾丸免疫豁免表征
				1. 生精细胞表达自身抗原， 但不诱导免疫反应
				2. 其他组织移植到睾丸， 免疫排斥若
				3. 睾丸抑制到其他组织， 耐受免疫排斥
			* 睾丸天然免疫系统： PRR（模式识别受体）
* 生精细胞
	* 类型
		* 精原细胞
			* 类型（根据位置）
				* A：
					* 最外层， 椭圆形
					* 通过有丝分裂增殖产生一个精原干细胞一个分化细胞
				* 中间型： 圆形
				- B型
					- 脱离基底膜， 有丝分裂停滞
					- 分化成精母细胞
				* 精母细胞
					* 尾部富含线粒体和高尔基体
					* 
				* 圆形精母细胞
				* 浓缩型精母细胞
# 减数分裂
* 初级精母细胞
	* 细线期， 偶线期， 粗线期， 双线期， 终变期
* 精细胞变形
	* 顶体形成
		- 高尔基体形成顶体泡，顶体泡融合形成顶体囊
		- 顶体囊内含顶体颗粒。
		- 顶体囊将水分排出， 形成双层膜，内含水解酶
		- 顶体面向管壁
		- 大型哺乳动物卵圆形， 小鼠大鼠镰刀形
	- 鞭毛
		- 中心粒微管迁移
		- 线粒体整合入鞭毛——线粒体鞘
			- 线粒体伸长， 迁移到中段， 围绕中央轴丝螺旋排列
		- 中央轴丝的外周有致密的纤维和纤维鞘
	- 体形变小
		- 多余胞质丢弃——残质体
		- 染色体凝缩， 精核重构（鱼精蛋白代替核小体与DNA结合， 大量， 非全部）——精子头
			- 鱼精蛋白是一种核精蛋白， 精膜上卵子结合蛋白合成
* 生精上皮周期： 两次分化程度相同的细胞群相继出现的时间间隔
* 生精上皮波： 生精细胞群沿生精管长轴的有序组织排列
# 精子的成熟
* 精子获能： 获取受精能力的过程
	* 雌性生殖道
	* 变化
		* 精子膜结构改变
		* 顶体凝集素结合区域发生改变， 可以识别精子受体

# 精子发生的调控
* 根据调节途径分为外源调节因素， 内源调节因素
* 外源调节因素
	* 激素
		* 睾酮： 精子发生
			* FSH： 支持细胞表面受体结合——分泌雄激素结合蛋白
				* 腺苷酸环化酶信号通路偶联受体， 诱导ABP等蛋白合成
			* LH: 刺激间质细胞——睾酮
				* 通过支持细胞或管周样细胞间接作用于生殖细胞
					* 间质细胞合成睾酮分泌到睾丸间隙液， 一部分进入生精小管与支持细胞合成的ABP结合
	* 旁分泌因子： 细胞产生的激素或调节因子通过细胞间隙调节临近细胞
		* IGF： 胰岛素生长因子
			* IGF-1促进LH与间质细胞结合， 促进睾酮分泌， 促进生精细胞DNA合成
		* TGF： 转移生长因子
			* TGF-α主要在间质细胞表达
			* 在支持细胞过量表达导致精子障碍
		* EGF
			* 支持细胞和生精细胞
			* 其低表达有利于维持精子发生
		* NGF： 
			* 生精细胞， 与减数分裂启动DNA合成有关
			抑制素：支持细胞
			* 抑制精原细胞和早期精母细胞DNA合成
		* Refinoic acid
			* RA调控减数分裂通路
* 内源调节因素
	* 精原干细胞和精源细胞增殖分化
		* SCF： 干细胞因子。作用于精原细胞分化扩增
		* kit： 原癌基因， 精原细胞内部分化的启动子（如A-B）
		* GDNF： sertoli cell分泌的神经营养因子——精原细胞扩增
	* 染色体动力学调控
		* SCP3： 联会复合体蛋白， 联会复合体的结构
			* 失活导致生精细胞凋亡， 不育
		* MSH4/5： 生精细胞特异性DNA错配修复蛋白， 参与染色体联会
			* 失活导致停滞在粗线期， 不育
	* 蛋白活性和稳定性调控
		* 泛素：标记蛋白被26S蛋白酶体降解

* 意义
* - 精子发生 ^8a3aae

	- PGCs进入生殖腺原基（发育成睾丸）， 进入性索停留。性索发育成生精小管（曲细精管）。管上皮细胞分化成支持细胞
	- 细胞质桥
		- Ad型精原细胞分裂开始，为细胞不完全分裂。
		- 细胞间通过细胞质桥连接， 形成合胞体， 进行物质和信息交流
		- 实现每群细胞同步成熟
	- 过程

			- 精子离开睾丸生精小管后形态结构成熟， 在附睾尾具有受精和定向运动能力
	- 调控机制
		- 激素调控
			- 下丘脑——促性腺激素释放激素（GnRH）
			- 脑下垂体——性腺激素（LH, FSH）
			- 睾丸
				- LS受体——生精小管间质细胞
					- 胆固醇转变为睾酮
						- 参与增殖， 减数分裂等过程
						- 没有睾酮无法完成精子发生
						- 肾上腺也能产生睾酮
				- FSH受体——生精小管支持细胞
					- 合成分泌蛋白为生殖细胞输送营养
					- 合成抑制素， 与睾酮进行负反馈调节
		- 旁分泌调节
			- 睾酮， 生长因子等
		- 基因调控
