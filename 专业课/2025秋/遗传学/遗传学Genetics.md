# 孟德尔比例的修饰
* 不完全显性
	* 亲本相对性状， 子代中间表型
* 共显性： 亲代性状全部在子代出现
* 复等位基因： 某基因在群体中具有三个及以上的等位基因形式
	* 血型
		* ABO
			* I： 凝集原
			* $I^A$: N乙酰半乳糖胺基转移酶——N-乙酰半乳糖胺
			* $I^B$: 半乳糖基转移酶——D-半乳糖
			* O: H物质
			* $i$: 假基因
		* 孟买血型
			* 孟买O型血： 没有H底物
			* $I^A, I^B, i, H, h$
		* 果蝇复眼基因
* 致死等位基因
	* 隐性致死： 纯合致死
	* 显性致死： 一个等位基因就会导致个体死亡
		* 亨廷顿病huntingtong disease
* 多基因效应
	* 多基因共同控制一个形状
* 一因多效
	* 一个基因控制多个性状
* 基因相互作用
	* 类型
		* 上位：一个基因影响另一个基因的表达
			* 示例： ABOH血型
			* 隐形上位基因： 隐性纯合体影响另一个基因表达
			* 显性上位基因：一个显性基因就能抑制其他基因表达
		* 互补：两个基因都是必需的
			* 互补分析（顺反测验）
				* 用于确定两个表型相似的突变是否位于同一基因（默认隐性突变）
				* 两个突变纯合体杂交
				* 情况1
					* 如果两个突变分属不同的基因： 出现互补——子代为野生型个体
				* 情况2
					* 如果两个突变属于同一个基因：不互补——子代仍然为突变型 
				* 互补群
					* 互补测验中结果不能互补的一系列突变——其属于同一基因内部的不同位点
		* 抑制： 一个基因抑制另一个基因的功能
		* 重复基因： 两个等位基因具有相同表型效果
		* 相互作用
* X连锁遗传：位于X染色体的基因
	* 性染色体： 与性别决定有关
	* 常染色体： 与性别无关
* Y连锁遗传：位于Y染色体的基因
* 限性遗传： 性状只出现在某一性状
* 从性遗传： 受到性别影响， 在两性中都可能出现
	* 由常染色体上基因控制的性状，在表现型上受个体性别影响的现象
	* 例如， Bb在雄性为显性， 在雌性为隐性
* 表型的表现
	* 表型收到基因型和环境共同调控
	* 外显率： 某种表型在群体中比例
	* 表现度不一： 表型的个体差异
	* 遗传早现： 表型在遗传过程中出现越来越早， 程度越来越重。
* 表观遗传学
	* DNA序列不变， 但基因表达模式改变引起表型变异
	* 表观遗传变异
		* DNA化学修饰
		* 组蛋白化学修饰
		* 染色质结构改变
* 基因组印记： 子代表型取决于该基因来自父母哪一方
* 母性影响： 母本基因型决定表型——基因表达时间性——细胞质
* 核外遗传
	* 细胞器遗传： 线粒体叶绿体DNA控制的性状的遗传方式

# 真核生物连锁与染色体作图
* 自由组合： 不同基因位于非同源染色体
* 连锁
	* 完全连锁： 基因位于同一染色体， 只能产生亲本配子
		* 果蝇杂交实验
	* 不完全连锁： 子代出现少数重组个体
		* 连锁互换定律的发展
			* 摩尔根的交换
				* 连锁的基因在染色体上线性排列
				* 任何两个基因都能发生交叉
				* 基因距离越远， 发生交叉次数越多
			* Sturtevant's principle
				* 交换率可以加和
				* 交换率可以代表基因间距离（非线性）
					* 图距单位1mu = 1cM = 1%重组率
				* 染色体作图
					* 基因交换类型
						* 单交换： 两个基因之间， 能检测到
						* 双交换： 两个基因之间， 不能检测到， 发生在三个基因之间能检测到
						* 三交换
					* 三点作图原则
						* 基因型为杂合体
						* 表现型对应基因型
						* 子代数量够多
					* 常染色体作图
						* 确定基因排序
							* 亲本基因型： F2中数量最多
							* 双交换基因型： F2数量最少
							* 单交换基因型： 剩余的
						* 确定基因之间的图距
						* 计算干涉与并发系数
							* 并发系数C = $\frac{观察到的DCOs}{预期的DCOs}
							* 干涉
								* I = 1-C
									* C = $\frac{观察到的双交换}{预期的双交换}$
								* 完全干涉
									* 观察不到双交换， C=0
								* 正干涉
									* I>0, 即0<C<1。观察到的双交换少于实际双交换
								* 负干涉
									* 即C>1。观察到的双交换>实际双交换
				* 交换会抑制相邻部位同时发生第二次交换。越近， 干涉越强
					* 染色体交叉重组角度
					* 染色体分带
						* 染色剂染色不均一
						* 需要标记内容：
							1. 染色体序号
							2. 臂序号
							3. 区序号
							4. 带序号

	* F因子：一种接合质粒
		* 控制F纤毛， 可以接合为供体菌提供遗传信息
	* F+菌： 具有F因子的细菌
		* 与受体菌交换基因的概率为1e-7
		* 接合后受体菌转化为F+
	* 
	* Hfr菌（高频重组菌）： F因子整合到了染色体中。概率为1e-3
		* 与受体菌交换基因的概率为1e-4
		* 整合位置处染色体基因先转移， F因子最后转移
			* 染色体作图
				* 中断杂交

# 群体遗传学
* 群体： 生活在同一地理区域的同一物种的具有杂交能力的一群个体
* 基因库： 群体中具有生殖能力的个体的所有等位基因的总和
* 群体基因频率的影响因素
	* 自然选择
		* 适合度： 对后代的遗传贡献
		* 选择系数： s = 1-w
			* 显性基因： $\Delta p = p'-p=\frac{-sp(1-p)^2}{1-sp(2-p)}$
			* 隐性基因： $\Delta q = q'-q=\frac{-sq^2(1-q)}{1-sq^2}$
			* 隐性致死：$q_g = \frac{q_0}{1+gq_0}$
				* g: generation
	* 突变: DNA或染色体发生改变的过程， 可产生新基因
		* 突变在进化过程中对等位基因频率的影响不显著
		* 突变率： 某个基因位点或群体中发生突变的概率
		* 突变对等位基因频率的影响:$\mu:A-a \hspace{2em} p_{g+1} = p_g - \mu p_g$
	* 基因流
		* 迁移： 个体从一个群体到另一个群体
			* $p_{i+1} = (1-m)p_i + mP_m$ 
	* 遗传漂变(抽样误差）
		* 群体越小， 基因频率波动越剧烈
		* 对杂合子的影响
			* 无限大群体： H = 2pq
			* 小群体：$H'= (1-\frac{1}{2N})H$
				* t代以后： $H_t = (1-\frac1{2N})^tH$
	* 不随机交配
		* 近交： 亲缘关系近的个体交配
			* 会导致杂合子减少
				* 后代比例： 
				* AA:$\frac{1-\frac1{2^n}}2$
				* Aa:$\frac1{2^n}$
				* aa:$\frac{1-\frac1{2^n}}2$
			* 近交系数： F——二倍体的两个等位基因来自同一共同祖先的概率
				* F=1： 群体中个体均为纯合体
				* F的计算
			* 近交衰退： 近交群体的适合度降低现象
* 基因型频率和等位基因频率计算
	* X连锁基因： 只需计算雄性的基因频率即可
	* 计算公式
		* p+q = 1
		* $(p+q)^2 = 1$
		* 卡方检验
			* 自由度
	* 平衡状态下， 基因型频率和等位基因频率保持不变
* 应用
	* 判断群体是否达到遗传平衡
	* 根据表现型频率计算等位基因频率
* hardy-weinberg平衡
