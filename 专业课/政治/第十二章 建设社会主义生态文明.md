## 一、总论
生态文明建设是关系中华民族永续发展的根本大计。十八大后党中央将生态文明纳入**五位一体**总体布局，形成习近平生态文明思想，生态环保实现历史性、转折性、全局性变化。思想核心概括为**十个坚持**。

十个坚持： 
	1. 坚持党对生态文明建设全面领导 
	2. 坚持生态兴则文明兴 
	3. 坚持人与自然和谐共生 
	4. 坚持绿水青山就是金山银山 
	5. 坚持良好生态环境是最普惠的民生福祉 
	6. 坚持绿色发展是发展观的深刻革命 
	7. 坚持统筹山水林田湖草沙系统治理 
	8. 坚持用最严格制度最严密法治保护环境 
	9. 坚持把美丽中国转化为全民自觉行动 
	10. 坚持共谋全球生态文明建设之路
# 第一节 坚持人与自然和谐共生
### （一）生态兴则文明兴
1. 人与自然是生命共同体：人类依托自然生存，破坏自然终将反噬自身。
2. 生态兴衰决定文明兴衰：古文明衰败、我国西北古代生态退化印证生态环境影响文明存续；中华文明自古拥有天人合一生态理念。
3. 生态文明是文明发展必然趋势：工业文明过度索取打破生态平衡，生态文明是人类文明前进方向；我国资源环境约束紧张，必须走生产发展、生活富裕、生态良好的发展道路。
### （二）绿水青山就是金山银山
1. 辩证关系：环保与经济对立统一，**在发展中保护、在保护中发展**；既要绿水青山，也要金山银山，宁要绿水青山不要金山银山，绿水青山就是金山银山。
2. 绿水青山多重价值**：兼具生态、自然、经济、社会财富，可转化生态产业实现多元收益。
3. 发展思路转变：依托生态优势实现高质量发展，严守生态红线。
### （三）生态文明建设提升至全局突出位置
1. 生态文明建设是重大的经济问题， 政治问题， 社会问题 
2. 生态文明战略地位更加凸显：生态文明入五位一体、基本方略、中国式现代化要求、强国目标、新发展理念、三大攻坚战；相关内容入党章、宪法。 
3. 制度体系更加完善：出台多项顶层文件，30 余部环保法律法规落地，中央环保督察常态化。
4. 全方位， 全地域， 全过程加强生态环境保护， 持续开展污染防治攻坚战。
5. 建设成效斐然：蓝天碧水净土攻坚成效突出，森林覆盖率、清洁能源占比等指标大幅优化，人居环境持续改善。

# 第二节 建设美丽中国
### （一）加快绿色生产与生活方式转型
1. 加快推动产业结构， 能源结构， 交通运输结构等调整优化
2. 推进各类资源节约利用， 转变资源利用方式， 提高资源利用效率。
3. 稳妥推进双碳：立足国情分步落实碳达峰碳中和，自主把控实施路径，提升生态碳汇。
4. 健全绿色发展的保障体系， 统筹各领域资源， 打好组合拳。
5. 把建设美丽中国转化为全体人民自觉行动
### （二）山水林田湖草沙一体化系统治理
1. 核心逻辑：山水林田湖草沙是有机生命共同体，不可分割的生态系统， 统筹生态全要素综合治理。
2. 重点举措
	1. 加快实施生态系统保护和修复重大工程。
    2. 构建以国家公园为主体的自然保护地体系（三江源、大熊猫等）建设
    3. 国土绿化、综合治理水土流失、荒漠化；
    4. 实施生物多样性保护工程；
    5. 落实河湖草原湿地休养生息。
### （三）以严格制度法治护航生态保护
1. 严格环保制度：构建源头预防 — 过程管控 — 损害追责全链条环保体系。
2. 高效资源利用制度：自然资源确权登记、资源有偿使用、垃圾分类、能源改革。
3. 生态追责制度：环保考核、领导干部自然资源离任审计、生态损害终身追责、河湖长制、林长制、环保督察。
4. 强化对破坏生态环境违法犯罪行为的查处抓办

# 第三节 共谋全球生态文明建设之路
### （一）守护人类共同地球家园
- 守护环境的必要性
	1. 地球是全人类赖以生存的唯一家园。
	2. 随着人口持续增长， 现代化进程推进， 地球的不可再生资源越来越少， 传统的发展模式难以为继
	3. 生态环境问题是全人类共同面临的问题
	4. 顺应自然， 保护自然， 绿色发展是全球可持续发展的大势所趋。
### （二）共建清洁美丽世界四大原则
1. 以人为本：环保结合减贫、就业，保障各国民生。
2. 科学治理：依托科技创新驱动全球绿色转型。
3. 坚持多边主义：反对单边主义，恪守国际环保公约。
4. 共同但有区别责任：发达国家承担历史排放责任，向发展中国家提供资金技术帮扶。
### （三）中国参与全球环境治理三大贡献
1. 气候治理：提前完成 2020 环保目标，人工林全球第一；承诺 2030 碳达峰、2060 碳中和。
2. 生物多样性：主办 COP15，促成《昆明 - 蒙特利尔框架》，落地多国生物多样性帮扶项目。
3. 绿色一带一路：搭建绿色国际联盟、环保大数据平台，落地南南气候合作项目。
