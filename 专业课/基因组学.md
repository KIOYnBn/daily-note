## 鸟枪法Shotgun Sequencing
- 一种高通量DNA测序方法
### 流程
1. 提取完整基因组 DNA
	- 从细胞里提取高分子量DNA（尽量长、不断裂）。
	- 目标：得到一整条/多条完整染色体DNA。
2. 随机打断 DNA（最关键一步）
	- 随机物理打断
	- 常用方法：
		- 超声波破碎（sonication）
		- 喷雾法、涡旋剪切
	- Sanger 测序：1–10 kb
	- NGS 二代测序：150–600 bp
3. 修复末端 + 加接头
	- 打断后的片段末端不整齐，要：
		- 用酶补平末端
		- 加上测序引物结合的接头（adapter）
4. 构建文库（Library）
	- 把小片段连到载体上（质粒、fosmid、测序接头），得到一大堆随机片段的集合
5. 大规模随机测序
	- 每个片段测出来的短序列叫：读段 / read
	- 测序深度（覆盖度）：一般要覆盖基因组 10×～100×，保证每个位置都被多次测到，减少错误。
6. 找重叠区（Overlap）: 电脑把所有 reads 两两比对，找序列完全一样的重叠部分。
7. 拼成连续序列 contig（连续叠连群）
8. 搭成 scaffold（支架）
	- 有些地方没有 reads 覆盖，会出现 gap（缺口）。用大片段文库（如 mate-pair）确定 contig 之间的顺序和方向 →连成 scaffold
9. 补洞与最终组装: 用 PCR、sanger 测序补 gap，最终得到接近完整的染色体/基因组序列。
### 鸟枪法的两种重要类型
1. 全基因组鸟枪法 WGS（Whole Genome Shotgun）
	- 直接把整个基因组打碎
	- 不做物理图谱、不分区
	- 速度快、成本低
	- 人类基因组计划后期就用这个
2. 分级鸟枪法（Hierarchical Shotgun）
	- 先把基因组切成很大的片段（BAC 克隆，~100–200 kb）
	- 构建物理图谱，确定顺序
	- 再对每个 BAC 做鸟枪法
	- 更稳，适合复杂基因组（重复序列多）
### 优点 & 缺点（考试常考）
- 优点
	- 速度极快，适合大规模测序
	- 不需要提前做遗传/物理图谱
	- 成本低、自动化程度高
	- 适合未知基因组、微生物组、宏基因组
- 缺点
	- 遇到大量重复序列（转座子、卫星DNA）容易拼错、拼乱
	- 组装复杂度高，依赖算法
	- 高度重复区域很难拼完整
六、和传统“步移法”对比
 
项目 鸟枪法 染色体步移 / 定向测序 
策略 随机打碎 → 随机测 → 电脑拼接 按顺序一段一段测 
速度 快 慢 
成本 低 高 
适合 大基因组、未知基因组 小片段、精确区域 
组装难度 高（重复序列麻烦） 低 
## DNA图谱
### 遗传图Genome Map
- 全称：遗传连锁图谱（genetic linkage map）
- 根据基因/标记之间「发生交换的概率」，画出它们在染色体上的「相对顺序和相对距离」。
#### 核心原理：连锁与交换
1. 连锁（linkage）
两个基因在同一条染色体上，离得近 → 一起传给后代的概率高 → 不容易分开。
2. 交换（crossing over）
减数分裂时同源染色体交叉互换，离得越远的基因，越容易被拆开 → 重组率越高。
#### 遗传图的单位：厘摩 cM
- cM（centimorgan，厘摩）
- 1 cM = 重组率 1%
- 距离近：0～5 cM（紧密连锁）
- 距离远：20～50 cM（几乎独立）
#### 步骤
1. 选遗传标记
以前用基因表型，现在多用 SNP、SSR 分子标记（已知位置的DNA小差异）。
2. 做杂交/家系： 看后代里，哪些标记总是一起出现，哪些经常分开。
3. 算重组率： 重组型个体数 / 总个体数 × 100% = 重组率（%）= cM。
4. 排顺序、算距离
5. 用三点测交等方法，确定：
	- 谁在左边
	- 谁在右边
	- 彼此隔多少 cM
6. 最后画出一条直线，上面标标记和cM距离 → 就是遗传图。
#### 遗传图有什么用？
1. 基因定位：找到致病基因、性状基因在染色体哪一块。
2. 辅助基因组组装：遗传图的顺序可以用来纠正测序拼接的错误。
3. 分子育种：标记跟性状连锁，不用等表型，直接选种子
### 物理图
- DNA片段在染色体上真实的物理位置、顺序和大小。
- bp、kb、Mb（碱基对、千碱基、兆碱基）
- 制作
	- 限制性酶切图谱
	- BAC/YAC克隆文库、重叠群（contig）
	- FISH 荧光原位杂交
### 序列图
- 完整的DNA碱基序列：A T C G 的精确排列
- 单位: 碱基对（bp）
- 制作： 鸟枪法测序 / 二代测序
### STS MAPING
STS Mapping（序列标签位点作图）：用“分子路标”拼基因组
 
STS Mapping = 用已知序列的DNA短片段（STS）当“坐标”，构建基因组物理图谱的技术，核心是PCR检测+重叠分析，把零散的DNA片段按真实顺序拼起来，是遗传图→物理图→序列图的关键桥梁。
 
 
 
一、先搞懂：STS是什么？
 
STS（Sequence-Tagged Site，序列标签位点） 是一段100–500 bp的DNA片段，必须满足两个条件：
 
1. 序列已知：能设计唯一引物对，用PCR特异性扩增
2. 在基因组中唯一：只出现一次，不会到处乱扩
 
常见STS类型：
 
- EST（表达序列标签）：来自mRNA的cDNA片段
- SSR/SSLP（微卫星）：重复序列长度多态性
- SNP：单碱基差异的位点
 
STS就像基因组上的**“门牌号”**，每个都有唯一位置，能被PCR精准识别。
 
 
 
二、STS Mapping的核心原理
 
一句话：两个STS如果总在同一段DNA片段上，它们在物理上离得近；越容易分开，离得越远。
 
- 用PCR检测大量克隆（如BAC、YAC文库）中哪些STS同时存在
- 统计共现频率，确定STS之间的顺序和物理距离
- 把STS作为“锚点”，将零散的DNA片段连成完整的重叠群（contig） 
 
 
 
三、完整实验流程（超详细）
 
1. 准备工作
 
- 构建大片段DNA文库（BAC/YAC，~100–200 kb）
- 设计多个STS的引物对（每个STS对应一对引物）
 
2. 核心步骤（PCR+重叠分析）
 
1. PCR筛选：用每对STS引物，逐一扩增文库中的克隆
- 出现条带 → 这个克隆含有该STS
- 无条带 → 这个克隆不含该STS
2. 重叠分析（关键）：
- 若两个STS同时出现在多个克隆中 → 它们在物理上靠得近
- 若两个STS很少共现 → 它们在物理上离得远 
3. 排序与连群：
- 按STS共现频率，确定谁在左、谁在右
- 把含重叠STS的克隆连成contig（重叠群）
- 计算STS间的物理距离（bp）
4. 构建物理图谱：
- 把contig按顺序排列，标上STS和真实bp距离 → 得到高分辨率物理图
 
3. 特殊方法：辐射杂种作图（Radiation Hybrid Mapping）
 
处理大基因组时，用X射线把细胞DNA打成碎片，融合到仓鼠细胞里，通过统计STS共现概率，计算断裂频率→物理距离，精度更高。
 
 
 
四、STS Mapping的“王牌优势”
 
1. 通用性：PCR检测简单、可重复，全球实验室都能用
2. 精确性：直接反映物理距离（bp），不是遗传图的概率（cM）
3. 可整合：能把遗传图（cM） 和物理图（bp） 精准对应，解决“遗传距离≠物理距离”的问题
4. 标准化：STS序列存数据库，任何人都能验证、复用 
 
 
 
五、和你学过的图谱怎么联动？
 
项目 遗传图 STS物理图 序列图（鸟枪法） 
依据 重组率（cM） STS共现+PCR（bp） 碱基序列重叠（bp） 
精度 低（区间） 中高（精确到kb） 最高（精确到碱基） 
关系 提供基因大概位置 用STS当“锚点”，纠正鸟枪法拼接错误 测出完整ATCG序列 
 
通俗比喻：
 
- 遗传图：告诉你“北京→上海→广州”的顺序
- STS物理图：告诉你“北京到上海1318km，上海到广州1780km”
- 鸟枪法序列图：告诉你“沿途每一公里的具体风景”
 
 
 
六、STS Mapping有什么用？
 
1. 基因组组装“导航”：人类基因组计划用它做**“序列就绪图”**，确保鸟枪法测序片段拼对顺序 
2. 基因定位“精准化”：把遗传图定位的基因区间，锁定到具体DNA片段上
3. 比较基因组学：用保守STS找不同物种的同源区域
4. 分子育种：把性状基因和STS连锁，精准选种
 
 
 
超精简版（背下来）
 
1. STS是已知序列的唯一DNA片段，能被PCR精准识别
2. STS Mapping = 用PCR查哪些克隆含相同STS → 确定片段重叠→拼出物理顺序 
3. 它是遗传图和序列图的桥梁，让“概率距离”变成“真实bp距离
## DNA Maker
### 一、遗传上的 DNA 标记（Genetic DNA Marker）
- 定义： DNA 上一些位置已知、能稳定遗传、个体之间有差异的短序列。
- 相当于染色体上的“路标”。
- 作用
	- 画遗传图
	- 做基因定位（找致病基因、性状基因）
	- 亲子鉴定、亲缘鉴定
	- 分子育种
- 常见类型
	- SNP（单核苷酸多态性）：单个碱基不一样
	- SSR / STR（微卫星）：重复单元数目不一样
	- RFLP：酶切位点不一样
### 实验电泳用的 DNA Marker（DNA Ladder）
- 做琼脂糖凝胶电泳时，加在旁边那条泳道的条带。
- 定义: 一管已知大小、已知 bp 的 DNA 片段混合物。
- 作用
	- 当作尺子
	- 看你样品的 DNA 是 多少 bp、多大
### 应用
- 在遗传图、基因组那章看到的，就是第一种（遗传标记）；
- 在PCR、电泳实验里看到的，就是第二种（分子量标准）。
