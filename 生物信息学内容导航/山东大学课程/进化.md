# 基本概念
- 分子进化理论： 在分子水平[DNA， RNA， 或蛋白质]研究进化过程
	- 基本假设： 
		1. DNA, RNA或蛋白质序列包含所有进化信息
		2. 分子钟理论： 一个特定蛋白质的进化变异速度在不同物种中基本恒定——序列越相近， 距离共同祖先越近
## 同源
- 同源[Homologs]: 来源于共同祖先的相似序列
- 直系同源[Orthologs]: 来源于物种形成中保留下来的基因
- 旁系同源： 同一物种中由基因复制产生的新基因
- 异同源： 通过水平基因转移产生的基因
# 系统发生树
- 概念： 根据不同物种的序列差异构建
- 作用
	1. 确定进化关系
	2. 预测序列功能
	3. 追溯基因起源
	4. 预测序列功能的走向
- 关键概念： 根， 枝， 节， 叶
	- 所有现存物种都在叶上
- 分类
	- 有根树
	- 无根树
	- `Note`： 二者可以相互转换
- 物种树： 基于物种进化关系构建
- 分子数： 基于基因或蛋白质序列关系构建
## 构建
### 原理
1. UPGMA： 基于距离的方法
	- 序列比对的一致度/相似度代表距离
2. MP： 最大简约法
	- 突变次数
3. ML： 最大似然法
4. Bayesian inference： 贝叶斯推断法
