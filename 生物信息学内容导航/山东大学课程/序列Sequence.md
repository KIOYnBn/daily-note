# 序列比对
* `序列比对的意义`: 相似序列很可能具有相似的空间结构和生物学功能
* `一致度`：若两序列长度相同，为对应位置相同残基（一个字母，氨基酸或碱基）的数目与长度之比
* `相似度`：若长度相同，相似或相同的残基数目与长度之比
	* 相似残基： 通过替换记分矩阵定义
		* 替换记分矩阵： 描述残基两两相似的量化关系
			* DNA替换记分矩阵
				* 等价矩阵： 相同核苷酸之间为1， 不同核苷酸之间为0（ps： 没啥用）
				* 转换颠换矩阵： 转换得分为-1， 颠换得分为-5
					* 转换： DNA碱基环数不变。即嘌呤或嘧啶内部转换
					* 颠换： DNA碱基环数变化。即嘌呤与嘧啶转换
				* BLAST矩阵： 相同核苷酸为5， 不同为-4
			* 蛋白质替换记分矩阵
				* 等价矩阵： 与DNA的等价矩阵相同。没用
				* PAM矩阵： 
					* 分类： PAM-n。PAM-1自乘n次
					* 示例： PAM-1
						* 相似度 > 85%的近缘蛋白多序列比对，统计每一百个氨基酸平均发生一个突变的量值
				* BLOSUM矩阵： 根据设定阈值先将相似序列归为一类。统计替换数目， 进行打分
				* 遗传密码矩阵： 根据替换氨基酸所需改变的密码子的数量
				* 疏水矩阵： 根据残基替换前后疏水性变化得到的矩阵。
* `蛋白质指纹(Prints)`： 一组保守的序列基序， 刻画了蛋白质家族的特征
	* PRINTS数据库

## 序列比对
### 双序列比对
* `原理`： 通过插入gap算法， 获得最优序列比对分数
- `类型`
	- 全局比对
	- 局部比对
- 常用软件： EMB
	- 重要参数： 
		- MATRIX: 替换记分矩阵算法
		- GAP OPEN： 起始gap的分值
		- GAP ENTEND： 连续GAP的分支
		- OUTPUT FORMAT: 
		- END GAP PENALTY： 是否设置结尾GAP赋值
		- END GAP OPEN： 结尾GAP赋值

### 数据库序列比对
* blast（Basic Loacl Aligment Search Tool)： 基本局部比对搜索工具
* `原理`： 先找出得分高的等长子序列， 随后子序列延长给出高分值片段对
* `类型`
	* 根据对象分
		* Blastp: 蛋白质序列搜索蛋白质序列数据库
		* Blastn: 核酸序列搜索核酸序列数据库
		* Blastx: 核酸序列按6条链翻译成蛋白质并搜索蛋白质序列数据库
		* tBlastn: 蛋白质序列搜索核酸序列数据库。数据库核酸序列翻译成蛋白质序列
		* tBlastx核酸序列按6条链翻译成蛋白质序列搜索核酸序列数据库
	* 按算法分
		* BLAST
		* PSI-BLAST: Position-Specific Iterated)位置特异性迭代BLAST
			* 通过位置特异性矩阵PSSM（Position-Specific Scoring Matrix)配合BLOSUM打分矩阵
				* PSSM: 每个位置出现的残基类型的比例
			* 用途： 寻找远源相似序列（大量突变）
		* PHI-BLAST: (Pateern-Hit Initiated)模式识别BLAST
			* 与输入序列相似并符合特定模式的序列
				* 正则表达式： 
					* {N}: 排除N
					* [ABC]: ABC中任意一个
					* N(3, 7)： N出现3-7次
					* x: 任意字符
				* 示例： {L}GEx[GAS]x(3, 7)
		* SmartBlast:BLASTp的简约版
			* INFO： 包含进化关系
		* 其他BLAST
			* WU-BLAST: 灵活
			* SSEARCH: 更准确
			* FASTA: 对于DNA序列更准确
			* BLAT： 小序列在大基因组中搜索
* `工具`
	* NCBI
		* 参数： 
			* DataBase： 搜索数据库
			* Organism： 选定物种
			* algorithm: 使用算法
	- ExPASy
	- Uniprot
	- DDBJ
### 多序列比对
- 作用
	- 寻找序列家族
	- 建立系统发生树
	- 寻找保守片段
	- 预测序列功能
	- 预测结构
- `Note`
	- 不能比对太多序列
	- 不能比对相似度过低的一组序列
	- 不能比对相似度过高的一组序列
	- 不能比对过短的序列
	- 不能比对具有重复域的序列
- 工具
	- Clustal(常用)
		- 结果分析
			- 非常保守： `*`表示
			- 相对保守： `:`表示
			- 仅保留了亲疏水性： `.`表示
			- Tree Data： 用发生树表示序列间关系
			- 一致度矩阵： 序列间的一致度关系
	- TCOFFEE
		- 功能
			- DNA， Protein， RNA序列比对
		- 特色： 蛋白质结构比对
	- MUSCLE
