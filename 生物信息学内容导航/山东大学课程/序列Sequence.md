# 序列比对
* 序列相似度
	* 作用： 很可能具有相似的空间结构和生物学功能
* 一致度：若两序列长度相同，为对应位置相同残基（一个字母，氨基酸或碱基）的数目与长度之比
* 相似度：若长度相同，相似或相同的残基数目与长度之比
	* 相似残基： 通过替换记分矩阵定义
		* 替换记分矩阵： 描述残基两两相似的量化关系
			* DNA替换记分矩阵
				* 等价矩阵： 相同核苷酸之间为1， 不同核苷酸之间为0（ps： 没啥用）
				* 转换颠换矩阵： 转换得分为-1， 颠换得分为-5
					* 转换： DNA碱基环数不变。即嘌呤或嘧啶内部转换
					* 颠换： DNA碱基环数变化。即嘌呤与嘧啶转换
				* BLAST矩阵： 相同核苷酸为5， 不同为-4
			* 蛋白质替换记分矩阵
				* 等价矩阵： 与DNA的等价矩阵相同。没用
				* PAM矩阵： 
					* 分类： PAM-n。PAM-1自乘n次
					* 示例： PAM-1
						* 相似度 > 85%的近缘蛋白多序列比对，统计每一百个氨基酸平均发生一个突变的量值
				* BLOSUM矩阵： 根据设定阈值先将相似序列归为一类。统计替换数目， 进行打分
				* 遗传密码矩阵： 根据替换氨基酸所需改变的密码子的数量
				* 疏水矩阵： 根据残基替换前后疏水性变化得到的矩阵。

## 序列比对
### 双序列比对
* 通过插入gap算法， 获得最优序列比对分数
- 类型
	- 全局比对
	- 局部比对
- 常用软件： EMB
	- 重要参数： 
		- MATRIX: 替换记分矩阵算法
		- GAP OPEN： 起始gap的分值
		- GAP ENTEND： 连续GAP的分支
		- OUTPUT FORMAT: 
		- END GAP PENALTY： 是否设置结尾GAP赋值
		- END GAP OPEN： 结尾GAP赋值

### 数据库序列比对
* blast（Basic Loacl Aligment Search Tool)： 基本局部比对搜索工具
	* 原理： 先找出得分高的等长子序列， 随后子序列延长给出高分值片段对
* 类型
	* 根据对象分
		* Blastp: 蛋白质序列搜索蛋白质序列数据库
		* Blastn: 核酸序列搜索核酸序列数据库
		* Blastx: 核酸序列按6条链翻译成蛋白质并搜索蛋白质序列数据库
		* tBlastn: 蛋白质序列搜索核酸序列数据库。数据库核酸序列翻译成蛋白质序列
		* tBlastx核酸序列按6条链翻译成蛋白质序列搜索核酸序列数据库
	* 按算法分
		* BLAST
		* PSI-BLAST
		* PHI-BLAST
* 工具： 
	* NCBI
