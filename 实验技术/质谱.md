
## 一、核心本质与基础原理
1. 核心定义： 通过测定离子的质荷比（m/z），实现物质精准定性、结构解析、绝对/相对定量
2. 用途： 测出分子的精确质量、氨基酸序列、修饰位点
3. 核心基础公式: $\boldsymbol{质荷比(m/z) = 离子的质量数(m) / 离子所带的电荷数(z)}$
	- 质量数m：单位为道尔顿(Da)，对于多肽/蛋白，1个氨基酸残基平均质量约110Da；
	- 电荷数z：离子带的正负电荷数，生物大分子电离后通常带正电，z≥1；
	- `Note`： 质谱通过检测这个数值，就能反推分子的精确质量。
 
## 核心工作流程
1. 电离（最关键的第一步）：将液态/固态的中性待测分子，转化为气相的带电离子（只有带电离子才能被电场操控、被检测）。生物大分子（蛋白、核酸）通过“软电离”技术实现完整电离
2. 质量分离：带电离子在电场/磁场中，根据m/z的差异发生分离
3. 检测：分离后的离子撞击检测器，转化为电信号，信号强度与离子数量成正比（对应物质的含量）。
4. 数据解析：将电信号转化为质谱图，横坐标为m/z，纵坐标为离子信号强度（丰度）
## 质谱仪的核心硬件组成
`离子源和质量分析器是决定质谱性能、适用场景的核心。`
### 进样系统: 
- 负责将待测样品稳定、无污染地送入离子源
- 直接进样：纯品样品直接进样，多用于小分子、纯蛋白鉴定；
- 联用进样：和分离技术联用，是生命科学的主流：
	- 液相色谱-质谱联用（LC-MS/MS）：先通过液相色谱将复杂蛋白/多肽/代谢物混合物分离，再依次送入质谱，解决复杂样品的分离问题；
	- 气相色谱-质谱联用（GC-MS）：多用于挥发性小分子、代谢物的分析；
- 靶板进样：专为MALDI离子源设计，样品和基质混合点在靶板上，送入质谱。
### 离子源
- 作用: 中性分子转化为气相带电离子
- 类型
	- 硬电离： 直接将大分子打碎， 无法检测完整分子
	- 软电离： 生物大分子（蛋白、多肽、核酸）

|        | 生命科学常用软离子电子源                              |                                            |
| ------ | ----------------------------------------- | ------------------------------------------ |
| 特性     | 电喷雾电离（ESI）                                | 基质辅助激光解吸电离（MALDI）                          |
| 电离方式   | 高压电场下，样品溶液在喷针末端形成带电液滴，溶剂蒸发后液滴碎裂，释放完整的带电离子 | 样品与过量基质混合结晶，激光照射基质，基质吸收能量后传递给样品，使样品完整解吸并电离 |
| 样品兼容性  | 兼容液相色谱联用（LC-MS/MS），适合复杂混合物、多肽、蛋白、代谢物      | 不兼容液相色谱，适合纯品、简单混合物、多肽、蛋白、微生物全细胞            |
| 电荷特点   | 可使大分子带多电荷，大幅降低m/z，能检测分子量高达100kDa以上的完整蛋白   | 主要使分子带单电荷/双电荷，m/z与分子量接近，适合分子量快速测定          |
| 核心应用场景 | 蛋白组学全谱分析、翻译后修饰检测、定量蛋白组学、代谢组学              | 蛋白/多肽精确分子量测定、微生物快速鉴定、多肽质量指纹图谱（PMF）鉴定       |
| 优势     | 联用能力强、灵敏度高、适合复杂样品、二级测序效果好                 | 操作简单、速度快、耐盐、通量高、成本低                        |

### 质量分析器（决定分辨率与精度）
- 作用： 按m/z差异分离带电离子，其性能直接决定了质谱的分辨率、质量精度、扫描速度。
#### 生命科学领域常用的质量分析器（按分辨率从低到高排序）：
##### （1）四极杆（Quadrupole, Q）
- 原理：通过四个平行电极的交变电场，仅允许特定m/z的离子通过，其余离子被过滤掉；
- 特点：结构简单、成本低、扫描速度快、定量稳定性极佳，分辨率低（低分辨质谱），无法区分质量差异极小的离子；
- 核心用途：三重四极杆（QQQ）用于靶向精准定量（MRM/PRM技术），是小分子、目标蛋白/多肽绝对定量的金标准。
 
##### （2）离子阱（Ion Trap, IT）
- 原理：通过电场将离子捕获在三维空间内，按顺序释放不同m/z的离子实现分离；
- 特点：体积小、灵敏度高、能实现多级质谱（MSⁿ），适合结构解析，质量精度和分辨率有限；
- 核心用途：小型化质谱、*目标物结构解析、de novo测序*。
 
##### （3）飞行时间（Time of Flight, TOF）
- 原理：离子在无场区的飞行管中飞行，m/z越小，飞行速度越快，到达检测器的时间越短，通过飞行时间计算m/z；
- 特点：质量范围无上限、扫描速度极快、分辨率高（高分辨质谱）、能实现精确质量测定；
- 核心用途：MALDI-TOF/TOF用于蛋白质量指纹图谱鉴定、多肽测序；Q-TOF用于非靶向蛋白组学、代谢组学。
- 公式： $K = \sqrt{\frac{L^2}{2V}}$
##### （4）轨道阱（Orbitrap）
- 原理：离子在纺锤形中心电极和外电极之间做轴向震荡，震荡频率与m/z成正比，通过傅里叶变换转换为质谱图；
- 特点：当前生命科学的金标准，超高分辨率、超高质量精度（质量偏差<1ppm，能测到分子量小数点后4-6位）、动态范围宽、灵敏度高；
- 核心用途：高深度蛋白组学、翻译后修饰位点精准鉴定、非靶向代谢组学、大分子结构解析。
##### （5）傅里叶变换离子回旋共振（FT-ICR）
- 原理：离子在强磁场中做回旋运动，回旋频率与m/z成正比，通过傅里叶变换得到质谱图；
- 特点：分辨率、质量精度全球最高，但仪器体积大、成本极高、操作复杂；
- 核心用途：顶级科研机构的超高精度结构解析、复杂混合物全组分分析。
### 4. 检测器与数据系统
- 检测器：常用电子倍增器、微通道板，将离子撞击的信号转化为可放大的电信号，*信号强度直接对应离子的数量（物质含量）*；
- 数据系统：将电信号转化为可视化的质谱图，通过内置算法、*数据库匹配*完成物质鉴定、定量、结构解析。
## 三、核心质谱技术分类与主流联用平台
## 1. 按质谱级数分类
（1）一级质谱（MS¹）
- 定义：仅做一次质量分离，直接检测所有离子的m/z和丰度；
- 输出结果：一级质谱图，横坐标为m/z，纵坐标为离子丰度；
- 核心用途：测定分子的精确分子量、物质初筛、定量分析。
（2）二级质谱/串联质谱（MS/MS）
- 定义：两次质量分离+一次离子碎裂，是蛋白/多肽序列分析、精准鉴定的核心技术；
- 核心工作流程：
	1. 第一次质量分离（一级质谱）：从所有离子中筛选出目标离子（称为母离子/前体离子）；
	2. 碰撞碎裂：将筛选出的母离子送入碰撞室，与惰性气体碰撞，发生肽键断裂，生成一系列碎片离子（子离子）；
	3. 第二次质量分离（二级质谱）：检测所有碎片离子的m/z和丰度，得到二级质谱图；
- 核心用途：多肽/蛋白的氨基酸序列测定、翻译后修饰位点定位、物质精准定性、靶向定量。
- 通俗比喻：一级质谱是“给所有分子称重，挑出你要的那个”，二级质谱是“把挑出的分子按肽键打碎，通过碎片拼接出完整的氨基酸序列”。
### 2. 生命科学领域主流联用平台
（1）MALDI-TOF/TOF质谱
- 组成：MALDI离子源+两级TOF质量分析器；
- 核心优势：操作简单、通量高、速度快、耐盐性好；
- 核心应用：
	- 微生物快速鉴定（临床检验科的核心应用，10分钟完成细菌/真菌鉴定）；
	- 蛋白/多肽精确分子量测定；
	- 多肽质量指纹图谱（PMF）蛋白鉴定（SDS-PAGE电泳切胶后，酶解为多肽，直接用MALDI-TOF鉴定蛋白身份）。
（2）三重四极杆（QQQ）质谱
- 组成：ESI离子源+三个串联的四极杆（Q1筛选母离子、Q2碰撞碎裂、Q3筛选子离子）；
- 核心优势：定量稳定性极佳、灵敏度高、动态范围宽；
- 核心应用：目标蛋白/多肽的绝对定量、小分子代谢物、药物浓度检测、临床标志物定量。
（3）四极杆-飞行时间（Q-TOF）质谱
- 组成：ESI离子源+四极杆+TOF质量分析器；
- 核心优势：兼具*四极杆的筛选能力*和TOF的高分辨、高质量精度；
- 核心应用：非靶向蛋白组学、代谢组学、多肽de novo测序、翻译后修饰初筛。
（4）Orbitrap系列高分辨质谱
- 主流型号：Q Exactive（四极杆+Orbitrap）、Orbitrap Fusion（四极杆+离子阱+Orbitrap）；
- 核心优势：超高分辨率、超高质量精度、超高灵敏度，能同时完成定性和定量；
- 核心应用：高深度全蛋白组学分析、翻译后修饰（磷酸化、泛素化等）位点精准鉴定、单细胞蛋白组学、非靶向代谢组学。
## 四、应用
1. 蛋白质/多肽的精准鉴定与序列分析
	- 核心技术路线：
		- 1. 蛋白样品（电泳分离后的胶条、溶液中的纯蛋白）通过胰蛋白酶酶解，切成特定的多肽片段（胰蛋白酶特异性切割赖氨酸K、精氨酸R的羧基端肽键，和你之前学的Arg对应）；
		2. 酶解后的多肽混合物通过LC-MS/MS检测，得到每个多肽的一级质谱和二级质谱图；
		3. 将二级质谱的碎片离子信息与数据库中理论蛋白的酶解多肽碎片进行匹配，实现蛋白的精准鉴定；
		4. 无数据库匹配时，可通过de novo测序，直接从二级碎片离子拼接出多肽的氨基酸序列。
2. 蛋白质翻译后修饰（PTM）的精准检测
	- 质谱可以通过精确的质量变化，检测到修饰的存在，还能通过二级碎片离子，精准定位修饰发生在哪个氨基酸残基上（比如确定某个Tyr位点是否发生磷酸化）；
	- 应用场景：细胞信号通路研究、肿瘤发生机制、蛋白功能调控研究。
3. 蛋白质的精准定量
	电泳仅能实现半定量，而质谱可以实现蛋白的相对定量和绝对定量，是蛋白组学差异分析的核心：
	- 相对定量：比较两组/多组样品中蛋白的表达量差异，常用技术：
		- Label-free（非标记定量）：直接通过质谱检测的肽段信号强度，计算蛋白的相对含量，操作简单、成本低；
		- iTRAQ/TMT（同位素标记定量）：用不同同位素标签标记不同样品的多肽，混合后一起上机检测，通过标签的信号强度比值计算蛋白差异，通量高、重复性好；
		- SILAC（体内同位素标记）：在细胞培养时用同位素标记的氨基酸（比如你之前学的Lys、Arg）掺入蛋白，实现体内标记，定量精度最高。
	- 绝对定量：通过加入已知浓度的同位素标记内标肽段，直接计算出样品中目标蛋白的绝对摩尔浓度，是蛋白绝对含量测定的金标准，常用技术为MRM/PRM。
4. 氨基酸与代谢物的定性定量分析
	- 氨基酸分析：质谱可以一次性精准定性、定量样品中20种标准氨基酸（包括你之前问的Val、Met、Arg、Tyr等），以及修饰氨基酸，灵敏度比传统氨基酸分析仪高2-3个数量级；
	- 代谢组学：通过GC-MS/LC-MS，一次性检测样品中几百到几千种小分子代谢物，实现差异代谢物筛选、代谢通路分析，是生命科学、医学研究的核心组学技术。
5. 与电泳的经典联用（你之前问的电泳+质谱）
	1. 蛋白样品通过SDS-PAGE/2D凝胶电泳分离，得到不同分子量的蛋白条带/点；
	2. 切下目标条带/点，进行胶内酶解（胰蛋白酶），将蛋白切成多肽片段；
	3. 提取多肽，通过MALDI-TOF或LC-MS/MS进行质谱检测；
	4. 数据库匹配，实现蛋白的精准鉴定，得到蛋白名称、序列、修饰等信息。
## 五、核心术语
1. 分辨率：质谱区分两个相邻m/z离子的能力，分辨率越高，越能区分质量差异极小的分子，比如Orbitrap的分辨率可达140000以上，而普通四极杆分辨率仅为几千。
2. 质量精度：质谱测定的m/z与理论值的偏差，单位为ppm（百万分之一），高分辨质谱的质量偏差通常<5ppm，能精准确定分子的元素组成。
3. 灵敏度：质谱能检测到的最低样品量，高端质谱能检测到飞摩尔（fmol）甚至阿摩尔（amol）级别的样品。
4. 动态范围：质谱能同时准确定量的最高和最低样品浓度的比值，动态范围越宽，越能同时检测到高丰度和低丰度的分子。
5. 肽质量指纹图谱（PMF）：蛋白酶解后得到的所有多肽的m/z集合，就像蛋白的“指纹”，通过与数据库比对，就能快速鉴定蛋白身份，是MALDI-TOF的核心应用。
