2D-DIGE（双向荧光差异凝胶电泳）
= 传统 2-DE 的荧光定量升级版
- 解决传统 2-DE 胶间差异大、定量不准、重复性差的问题。
## 原理
用**三种荧光染料（Cy2/Cy3/Cy5）**分别标记：
- Cy3：实验组蛋白
- Cy5：对照组蛋白
- Cy2：所有样品混合的内参（Internal Standard）
然后三者混在同一块胶做 2-DE，电泳后分波长扫描，得到三张图，直接在同胶内定量比较。
## Cy 染料特点
- 分子量、电荷完全匹配：标记后蛋白 pI、分子量几乎不变 
- 只标记少量 Lys 残基（亚摩尔标记）：不影响迁移行为
- 光谱完全分开：Cy2（蓝）、Cy3（绿）、Cy5（红）可单独扫描 
## 流程
1. 样品标记- 实验组：Cy3（绿）
- 对照组：Cy5（红）
- 内参（所有样品混合）：Cy2（蓝）
2. 三样品等量混合 → 上同一块胶
3. 第一向：IPG-IEF（pI 分离）
4. 第二向：SDS-PAGE（分子量分离）
5. 激光扫描- 扫 Cy2 → 内参图
- 扫 Cy3 → 实验组图
- 扫 Cy5 → 对照组图
6. 软件分析- 以内参为基准点对点匹配、归一化
- 直接算：实验组 / 对照组 = 表达倍数 
## 与2-DE相比的优点
- 传统 2-DE：
对照组一块胶、实验组一块胶 → 胶间差异大、匹配难、定量不准
- DIGE：
三组在同一块胶 → 完全相同电泳环境 → 无胶间误差、定量极准
## 核心优势
 
1. 消除胶间差异：同胶跑样，重复性飞跃
2. 精准内参校正：Cy2 内参实现绝对归一化
3. 灵敏度高：荧光检测比考染、银染更灵敏、线性范围宽 
4. 直接定量：同点三通道强度比 = 表达比 
5. 减少胶数量、省时间
 ## 缺点
- 染料成本高
- 只能标记赖氨酸，极个别蛋白标记不到
- 仪器要求：激光扫描仪
