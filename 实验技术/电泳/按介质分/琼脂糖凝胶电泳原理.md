
# 琼脂糖凝胶电泳（Agarose Gel Electrophoresis）
- 作用： 分离、鉴定、纯化 DNA/RNA 片段最常用的方法。
- 利用电场，让不同大小的核酸分子在多孔的琼脂糖凝胶中移动速度不同，从而分开。
## 材料
1. 琼脂糖（Agarose）
	- 从海藻里提取的多糖
	- 加热溶解、冷却后 → 形成多孔网状凝胶
	- 像筛子
2. 核酸（DNA/RNA）
	- DNA/RNA 骨架上有 磷酸基团
	- 在 中性或弱碱性 pH 下：带负电（-）
3. 电泳缓冲液（如 TAE、TBE）
	- 导电
	- 维持 pH
	- 保护核酸不被降解
4. 染料（EB、GoldView、SYBR Green 等）
	- 插入 DNA 双链
	- 紫外灯下 发光，用来看见条带
## 核心原理
1. 电荷原理：DNA 带负电，向正极跑
2. 分子筛效应：凝胶像“筛子”
	- 琼脂糖冷却后形成 三维网状小孔。
	- 片段小的 DNA： 孔容易穿过去 → 跑得快
	- 片段大的 DNA：阻力大、难穿过孔 → 跑得慢
3. 分离的决定性因素：分子量（片段大小）
	- 在琼脂糖电泳中：
		- 相同构象（都是线性双链 DNA）
		- 相同电压、相同凝胶浓度
	- 和 DNA 碱基序列基本无关。
4. 电压、凝胶浓度的影响
	1. 凝胶浓度
		- （1）琼脂糖浓度越高
			- 孔越小
			- 分离 小片段 DNA 效果越好
			- 大片段难移动，分辨率变差
		- （2）琼脂糖浓度越低
			- 孔越大
			- 适合分离 大片段 DNA
		- 常用浓度：
			- 0.8%：大片段（500 bp ~ 15 kb）
			- 1.0% ~ 1.5%：普通 PCR 产物（100 ~ 3000 bp）
			- 2% ~ 3%：小片段、微小差异
	- （3）电压
		- 电压高 → 跑得快，但条带易模糊、发热、降解
		- 电压低 → 跑得慢，条带清晰、分辨率高
## 流程
1. 制胶
	- 琼脂糖 + 缓冲液 → 加热溶解 → 倒胶 → 插梳子 → 冷却凝固 → 拔梳子形成加样孔
2. 放胶、加缓冲液
	- 凝胶放入电泳槽，淹没凝胶
3. 样品与上样缓冲液混合
	- 上样缓冲液：
		- 增加密度 → 让样品沉到孔底
		- 有色指示剂（溴酚蓝）→ 看跑到哪了
4. 加样： DNA 样品加入加样孔
5. 通电电泳
	- 负极在加样孔一端
	- 正极在另一端
6. 染色/观察
	- 紫外灯下观察：
		- 亮带 = DNA
		- 位置 = 片段大小
## 影响迁移率的其他因素
1. DNA 构象
	- 超螺旋质粒：跑得最快
	- 线性 DNA：中间
	- 开环质粒：最慢
2. 离子强度
	- 缓冲液离子高 → 导电好、跑得快
	- 离子太低 → 跑很慢
3. 温度： 太热 → 凝胶融化、DNA 变性
4. 嵌入染料： 染料结合 DNA 会略微减慢迁移
