
## 一、基本定义
- 二代测序 = NGS（Next Generation Sequencing）
- 又称：高通量测序、大规模并行测序
- 代表平台：Illumina（最主流）、Thermo Ion Torrent、BGI华大
- 特点：摆脱Sanger「单次1条序列、逐个测」的模式，同时上亿条DNA片段同步测序，超高通量、低成本、全基因组/转录组全覆盖。
## 二、和一代Sanger 核心区别
1. Sanger（一代）
	- 单条片段依次测，读长长（800~1000bp），准确率极高，通量极低、贵
	- 适合：单基因、片段验证、克隆测序
2. NGS（二代）
	- 海量片段并行测序，通量爆炸、极便宜，读长偏短（150/300bp）
	- 适合：全基因组、转录组、微生物宏基因组、肿瘤测序
## 三、Illumina 测序原理
1. 两大关键技术： 桥式PCR扩增 + 可逆终止荧光测序（合成测序法）
### 步骤拆解
- ① 建库（文库构建）
	1. 将基因组DNA随机打断成短片段（200~350bp）
	2. 两端加上通用接头（Adapter）
	3. 加上Index（标签），可以多样本混测
	- 目的：把长DNA切成短片段，加上固定序列，才能结合测序芯片
- ② 芯片固定 + 桥式PCR
	1. 测序流动槽（Flow Cell）表面布满固定引物
	2. 文库片段两端接头与芯片引物互补，发生桥式结合
	3. PCR原位扩增，一个片段 → 一簇相同克隆（DNA簇）
	- 作用：放大信号，单条DNA信号太弱，簇才能被仪器捕捉
- ③ 边合成边测序（核心反应）： 使用4种荧光标记、可逆终止dNTP：
	1. 每一轮只加1个碱基
	2. 不同碱基（A/T/C/G）带不同荧光
	3. 结合后终止链延伸，拍照扫描荧光
	4. 化学切除终止基团与荧光，进入下一轮
- 循环往复，逐碱基读取序列。
- ④ 数据分析： 仪器拍下每一轮荧光信号 → 拼接每一段短读长 → 生物信息学比对、组装、变异分析。
## 四、二代测序通用完整流程
1. 样本处理：DNA/RNA提取、质控
2. 片段化：超声/酶切打断
3. 文库构建：加接头、加index、PCR富集
4. 簇生成：Flow Cell原位扩增
5. 循环测序：逐碱基荧光拍照
6. 下机数据分析：质控→比对→组装→变异/表达分析
## 五、二代测序优缺点
- 优点
1. 超高通量：一次运行测人类全基因组
2. 成本极低：相比Sanger，单位碱基成本下降万倍
3. 无偏好全覆盖：可测全部基因、非编码区、微生物
4. 适用范围极广：DNA、RNA、甲基化、ChIP等
- 缺点
	1. 单条读长较短（150/300bp），重复序列组装困难
	2. 有系统误差，单碱基准确率低于Sanger
	3. 数据分析量大，依赖生信服务器与算法
## 疑难点
### index，adaptor如何加入到基因片段
一、核心名词
1. 接头 Adapter
	- 本质：一段人工设计的短双链DNA序列
	- 不是样品本身的序列
	- 作用
		1. 让打断的随机DNA片段能结合到测序芯片（Flow Cell）上；
		2. 提供PCR引物结合位点，用来扩增文库；
		3. 提供测序引物结合位点，仪器才能启动读序列。
2. 标签 Index（也叫Barcode）
	- 本质：加在接头里的一段短小特异序列
	- 每个样本对应一段独一无二的Index。
	- 核心作用：多样本混测
- 一段建好的NGS文库片段完整结构：测序引物区 — Index(标签) — 通用接头 — 样品目的DNA — 反向通用接头 — Index
### 接头 & Index 如何添加？
- 步骤1：DNA片段化： 基因组DNA → 超声打断/酶切打断 → 变成200~350bp短片段。
- 步骤2：末端修复 + 加A尾（关键前置）
	1. 打断后的DNA末端参差不齐，用Taq酶进行*末端补平*， 在3'端加一个*A碱基*；
		- 目的：为了和接头 T-A连接，提高连接效率。
- 步骤3：T-A连接法 — 连接接头（核心步骤）
	- 商用接头本身3'端带一个 T碱基；
	- 利用 A-T互补，在T4连接酶作用下：
	- 样品DNA（3'-A） ⇋ 接头（3'-T）
	- 直接共价连接，把接头+Index一次性连在片段两端。
	- Index 是直接设计在接头序列里的
- 步骤4：文库PCR扩增： 利用接头上的通用序列作为引物，PCR扩增；
- 步骤5：文库质控、混样上机： 不同样本 → 不同Index → 按比例混合 → 一起上测序仪。
### 可逆终止dNTP是啥
① Sanger 用的：ddNTP（双脱氧核苷三磷酸）
- 天然dNTP：3'-OH ✅
- ddNTP：3' 完全没有 OH（3'-H）
- 结果：只要掺入→永久不可逆终止，链再也接不上下一个碱基。
② NGS  illumina 用的：可逆终止 dNTP
本质还是 正常脱氧核苷三磷酸：
- 3'-OH 被化学基团封闭遮挡（不是直接删掉）
- 碱基上连有特异性荧光基团
- 关键：测序拍完照后，可以用化学试剂切掉封闭基团+荧光——3'-OH 恢复，链继续延伸
