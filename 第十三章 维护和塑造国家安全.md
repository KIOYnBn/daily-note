# 二、第一节 坚持总体国家安全观
### （一）国家安全是民族复兴的根基
1. 国家安全定义： 指国家政权、主权、统一和领土完整、人民福祉、经济社会可持续发展和国家其他重大利益，处于无危险、不受内外威胁的状态，以及保障持续安全状态的能力。
2. 核心地位： 国家安全是民族复兴的根基，社会稳定是国家强盛的前提，是党治国理政、国泰民安的基础性、根本性工作。
3. 新时代国家安全复杂形势。新时代国家安全内涵外延、时空领域、内外因素空前复杂，风险具有联动性、传导性、叠加性。
### （二）总体国家安全观的核心内涵与定位
1. 提出时间: 2014年4月15日，中央国家安全委员会第一次会议首次提出，是新时代国家安全工作的基本遵循。
2. 核心要义（五大维度）
	- 以人民安全为宗旨：一切为了人民、一切依靠人民，保障人民根本安全利益。
	- 以政治安全为根本：首要位置，保障政权、制度、意识形态安全。
	- 以经济安全为基础：夯实产业、科技、资源、金融等领域安全，以发展筑牢安全根基。
	- 以军事、科技、文化、社会安全为保障：软硬实力结合，全方位筑牢安全屏障。
	- 以促进国际安全为依托：践行全球安全观，共建普遍安全的人类命运共同体。
3. “总体”的核心逻辑（大安全理念）
	覆盖政治、军事、网络、粮食、生态、数据、人工智能等全领域安全；
	坚持系统统筹，统筹内外安全、国土与国民安全、传统与非传统安全、自身与共同安全，统筹维护与塑造国家安全；
	贯穿党和国家工作全过程、各方面，凝聚全社会整体合力。
- 总结为总体国家安全观“十个坚持”
	1. 坚持党对国家安全工作的绝对领导
	2. 坚持中国特色国家安全道路
	3. 坚持以人民安全为宗旨
	4. 坚持统筹发展和安全
	5. 坚持把政治安全放在首要位置
	6. 坚持统筹推进各领域安全
	7. 坚持把防范化解国家安全风险摆在突出位置
	8. 坚持推进国际共同安全
	9. 坚持推进国家安全体系和能力现代化
	10. 坚持加强国家安全干部队伍建设
### （三）新时代国家安全建设重大成就
1. 党的领导全面强化：成立中央国安委，出台专项条例，落实国家安全责任制，形成集中统一、高效权威的领导体制。
2. 制度体系日趋完善：以《国家安全法》为核心，出台反间谍法、网络安全法、数据安全法等系列法规，完善国家安全战略体系。
3. 核心利益全面维护：坚决抵御外部打压遏制，稳边固边，打击分裂势力，捍卫国家主权、安全、发展利益。
4. 平安中国提质升级：扫黑除恶常态化，社会治安持续向好，我国成为全球最具安全感的国家之一。
5. 全民防线全面筑牢：设立4·15全民国家安全教育日，安全教育纳入国民教育体系，全民安全意识显著提升。

# 第二节 构建统筹各领域安全的新安全格局
### （一）核心原则：统筹发展和安全
1. 战略定位： 统筹发展和安全是党治国理政的**重大原则**，写入“十四五”指导思想、党的二十大党章。
2. 二者辩证关系
	- 发展是基础：发展是解决安全问题的总钥匙，是国家存续发展的根本支撑。
	- 安全是保障：无安全则无发展，安全是发展的前提，保障发展成果不被损毁。
	- 双向赋能：以发展促安全、以安全保发展，实现高质量发展与高水平安全良性互动。
3. 核心目标： 以新安全格局保障新发展格局，增强国家生存力、竞争力、发展力、持续力，保障民族复兴进程不被迟滞、中断。
### （二）首要任务：把政治安全放在首位
政治安全是国家安全的根本，决定其他所有领域安全，核心是政权安全、制度安全、意识形态安全。
1. 政权安全：坚持党的全面领导，巩固党的长期执政地位。
2. 制度安全：坚定制度自信，坚守中国特色社会主义制度，抵御西方制度渗透颠覆。
3. 意识形态安全：巩固马克思主义指导地位，牢牢掌握意识形态工作领导权、管理权、话语权，筑牢思想防线。
三者统一：政治安全是根本保障，人民安全是核心宗旨，国家利益至上是根本原则。
### （三）重点领域国家安全（主战场）
1. 国土安全（立国之基）：捍卫领土、领海、领空主权，反对“台独”“藏独”“东突”等分裂势力，打击三股势力，维护边疆稳定和港澳繁荣稳定。
2. 经济安全（基础支撑）：保障基本经济制度、产业链供应链安全；防控金融、债务系统性风险；守住18亿亩耕地红线、保障粮食安全；强化核心技术自主可控，保障能源资源稳定供给。
3. 社会安全（民生底线）：平安是基本民生，常态化扫黑除恶，完善立体化治安防控，化解社会矛盾，保障群众生命财产安全。
4. 网络、AI、数据安全：没有网络安全就没有国家安全；强化网络综合治理、AI安全监管、关键数据保护，筑牢数字安全屏障。
5. 生物与公共卫生安全：健全疫情防控、生物风险预警体系，防范重大传染病、生物入侵等风险，守护公共卫生安全。
6. 外部安全与海外利益安全：参与全球安全治理，深化国际安全合作，构建海外利益保障体系，保护海外公民、机构合法权益。

# 第三节 开创新时代国家安全工作新局面
### （一）推进国家安全体系和能力现代化
是国家治理体系和治理能力现代化的重要内容，是应对复杂安全风险的必然要求。
1. 健全国家安全体系： 完善领导体制、法治体系、战略体系、风险预警、应急管理、重点领域安全保障体系；健全反制裁、反干涉、反“长臂管辖”机制，构建全域联动、立体高效的安全防护体系。
2. 增强国家安全能力： 强化科技赋能、法治思维、基层基础，提升重点领域、海外安全保障能力；常态化开展国家安全教育，筑牢全民人民防线。
### （二）建设更高水平的平安中国
平安中国是现代化建设的重要保障，聚焦治理更高效、局势更稳定、人民更满意三大目标：
1. 治理效能升级：坚持预防为主，公共安全治理从事后处置转向事前预防，强化安全生产、防灾减灾、食品药品安全监管。
2. 风险源头管控：多元化解社会矛盾，完善信访、网格化治理机制，把风险化解在萌芽状态。
3. 民生平安提质：深化基层平安创建，精准打击违法犯罪，全方位保障群众合法权益。
### （三）提高防范化解重大风险能力
1. 坚持底线思维、极限思维：做足最坏打算、争取最好结果，加固安全底板，补齐发展短板。
2. 统筹应对各类风险：高度警惕“黑天鹅”、防范“灰犀牛”，主动预判风险、精准化解风险。
3. 发扬斗争精神：直面风高浪急的考验，主动识变、应变、求变，打好风险防范主动战。
4. 健全风险防控机制：完善研判、评估、协同、责任落实机制，以制度威力抵御风险冲击。

# 课后思考题（核心答题要点）
1. *如何认识新时代我国国家安全形势的新变化？* 核心：内外风险交织、领域更广、因素更复杂、风险传导更快；由高速发展转向高质量发展阶段，由大向强高风险期，传统+非传统风险叠加。
2. *如何理解总体国家安全观的丰富内涵？* 核心：立足“十个坚持”，把握五大核心维度、大安全系统思维、统筹发展与安全、全民参与、内外兼顾的中国特色国家安全道路。 
3. *为什么说统筹发展和安全是党治国理政的重大原则？* 核心：发展是根本动力，安全是根本保障，二者一体两翼；是应对复杂形势、保障民族复兴、实现长治久安的必然要求。 
4. *为什么要把维护政治安全放在首要位置*核心：政治安全是国家安全的根本，决定所有领域安全，关乎党的执政地位、国家制度、意识形态根基，是国泰民安的前提。
