-  `蒸馏`（Knowledge Distillation）：目标是把大模型变小。
- **目的**：你有一个性能很好的“教师模型”（很大，推理慢），你想训练一个“学生模型”（很小，推理快）来模仿教师的行为，最终**替换掉**教师模型。
- 实现
```python
import torch.nn.functional as F

def distillation_loss(student_output, teacher_output, true_labels, temperature=4.0, alpha=0.7):
    """
    计算蒸馏损失
    Args:
        student_output: 学生模型的原始输出 (logits)
        teacher_output: 教师模型的原始输出 (logits), 不参与梯度计算
        true_labels: 真实的硬标签
        temperature: 温度参数 T, 用于软化概率分布
        alpha: 平衡硬标签损失和软标签损失的权重
    """
    # 1. 硬标签损失：学生与真实答案的差距 (标准交叉熵)
    hard_loss = F.cross_entropy(student_output, true_labels)

    # 2. 软标签损失：学生与教师思考过程的差距 (KL散度)
    # 使用温度T软化教师的输出，作为学生的“模仿目标”
    soft_teacher = F.softmax(teacher_output / temperature, dim=1)
    # 学生的输出也要用同样的温度软化后，再去与教师比较
    soft_student = F.log_softmax(student_output / temperature, dim=1)
    soft_loss = F.kl_div(soft_student, soft_teacher, reduction='batchmean')
    # 乘以T^2是为了平衡梯度尺度，使损失不受温度影响过大
    soft_loss = soft_loss * (temperature ** 2)

    # 3. 总损失 = 加权和
    total_loss = alpha * hard_loss + (1 - alpha) * soft_loss
    return total_loss
```
